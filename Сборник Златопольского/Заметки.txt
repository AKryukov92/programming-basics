Начиная с задания 1.14 нужны переменные числовых типов и конвертирование.
Точность задач, в которых получаются десятичные числа - четыре знака после запятой.

Задание 1.27 решается с помощью теоремы пифагора. Касательная к Земле от данного отрезка будет под прямым углом к радиусу.
В задании 1.31 среднее геометрическое это корень n степени из произведения n чисел.
Для двух чисел - корень квадратный. Для трех чисел - корень третьей степени.
Чтобы не было переполнения, можно считать произведение корней n степени.

В задании 1.34 нужно выразить x через остальные переменные.

В задании 1.36 можно взять абсолютное значение разницы любых радиусов.

Задание 1.38 решается за несколько вычислений.
Сначала ищем разницу между основаниями. Считаем эту величину основанием равнобедренного треугольника.
Затем ищем боковую сторону этого треугольника по теореме пифагора используя данную высоту и вычисленную длину основания.
Периметром трапеции будет сумма оснований и удвоенной длины боковой стороны.
