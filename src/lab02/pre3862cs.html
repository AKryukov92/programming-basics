К вступлению

В примере ниже, сообщение будет склеено из фрагментов строк и значений переменных.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String x; x = Console.ReadLine(); String y; y = Console.ReadLine(); Console.Write("<point x=\"" + x + "\" y=\"" + y + "\">"); } } }

В следующем примере, сообщение будет описано шаблоном. В нем с помощью символьных комбинаций {0} и {1} будут отмечены места для записи переменных. А после шаблона через запятую указаны переменные, значения которых нужно подставить на место этих комбинаций.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Task3862 { class Program { static void Main(string[] args) { String x; x = Console.ReadLine(); String y; y = Console.ReadLine(); Console.Write("<point x=\"{0}\" y=\"{1}\">", x, y); } } }

Переменная, указанная сразу после шаблона, получила номер 0 и была подставлена на место {0}. Следующая переменная - 1 и она была подставлена вместо {1}. Можно указывать множество переменных. Третья переменная получит номер 2 и так далее.

На следующем шаге команда форматированного вывода будет применена для вывода чисел с нужной точностью.

Шаг 2/3

К шагу 1 К вступлению

По условию задачи нужно вывести значения π и e. Вы, конечно, можете найти в интернете значения этих математических констант и написать их вручную в код программы. Но намного проще применить встроенные в язык константы "Math.PI" и "Math.E". Если использовать их, то при опечатке, вы сразу же о ней узнаете, и программа не запустится. Опечататься в одной из цифр длинного числа намного легче, и при этом ошибка будет не заметна.

Выведите в консоль сообщение, содержащее значение этих констант.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Task3862 { class Program { static void Main(string[] args) { Console.WriteLine("{0}\n{1}", Math.PI, Math.E); } } }

В результате получится

3,14159265358979 2,71828182845905

Обратите внимание, что возле Math.PI и Math.E не нужны ". Эти значения постоянны, но технически реализованы как переменные с именами и запретом на изменение.

Теперь нужно ограничить эти значения четырьмя символами после запятой. Для этого нужно заменить {0} на {0:F4}. Число "4" тут значит количество отображаемых символов после запятой. Более подробную справку о форматированном выводе можно посмотреть на msdn (открыть в новой вкладке)

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Task3862 { class Program { static void Main(string[] args) { Console.WriteLine("{0:F4}\n{1:F4}", Math.PI, Math.E); } } }

В результате получится

3,1416 2,7183

К вступлению

Вступление

В предыдущем разборе теории (открыть в новой вкладке) была рассмотрена команда для считывания значений, введенных пользователем в консоль.

Сейчас будет описано как выводить на экран специальные значения чисел.

Возможностей простой команды вывода в консоль может не хватить в двух случаях:

  • Вывод сложных сообщений, состоящих из множества фрагментов и значений переменных.
  • Вывод чисел с определенным количеством знаков после запятой.

В этих случаях полезно использовать команду форматированного вывода. Она состоит из технического оформления, шаблона сообщения и списка переменных. Самый простой пример оформления такой команды указан ниже.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String data; data = Console.ReadLine(); Console.Write("{0}", data); } } }
