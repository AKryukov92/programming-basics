Шаг 1/3

К вступлению

Для хранения чисел в языке создано множество типов переменных, которые можно разделить на две группы:

  • Для целых чисел. Например int. Это те числа, которыми мы пользуемся для нумерации чего-нибудь или подсчета количества.
  • Для действительных чисел. Например double. Они нужны для измерения массы, расстояния, площади и других величин. Действительные числа еще называют дробными. Числа π и e являются действительными.

Ниже показан пример объявления переменной для дробного числа. Она имеет тип "double" и называется "a".

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String data; data = Console.ReadLine(); double a; } } }

Предположите как нужно объявлять переменную для целого числа. Допустим ее имя будет называться "b". Напоминаю что для хранения целых чисел вам потребуется тип "int".

Шаг 2/3

К шагу 1 К вступлению

Просто объявить переменную для чисел мало. Нужно записать в нее значение, введеное пользователем. Попробуйте написать тривиально, как во фрагменте ниже.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String data; data = Console.ReadLine(); double a; //Это вариант для иллюстрации ошибки! a = data; } } }

При попытке запустить программу, вы должны увидеть сообщение об ошибке.

Ошибка: Не удается неявно преобразовать тип "string" в "double".

Фрагмент кода выше был приведен для того, чтобы познакомить вас с сообщением об ошибке. Для ее исправления нужно воспользоваться командой явного преобразования.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String data; data = Console.ReadLine(); double a; //Это корректный вариант! a = double.Parse(data); } } }

Ошибка исправлена, программа должна компилироваться успешно. Теперь нужно вывести значение этой переменной на экран.

Шаг 3/3

К шагу 2 К вступлению

Для вывода значения числовой переменной достаточно просто написать команду вывода как в этом фрагменте кода.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String data; data = Console.ReadLine(); double a; a = double.Parse(data); Console.WriteLine(a); } } }

Кроме того, вы можете воспользоваться командой форматированного вывода. Более подробно она была описана в предыдущем разборе теории (открыть в новой вкладке).

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String data; data = Console.ReadLine(); double a; a = double.Parse(data); Console.Write("Значение a: {0:F4}", a); } } }

Вступление

В предыдущем разборе теории (открыть в новой вкладке) был рассмотрен способ вывода числа π с нужной точностью.

Сейчас будет разобран ввод и вывод чисел с клавиатуры.

Вспомните как оформлялась команда получения исходных данных ранее.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String data; data = Console.ReadLine(); } } }

Обратите внимание на то, что в этом случае вы получаете текст. Даже если пользователь введет "17", то компьютер воспримет это как две буквы, а не как число 17. При этом переменные строкового типа String для вычислений не подходят. Что может получиться при умножении слова на букву? Вряд ли что-то полезное. Операция умножения имеет смысл для чисел, а не для слов и букв.
