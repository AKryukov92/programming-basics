Задача

Переведите алгоритм, описанный ниже, с естественного языка на ваш язык программирования.
Объявите переменные:
  1. temp для временного хранения строк получаемых от пользователя
  2. height для хранения высоты прямоугольника
  3. width для хранения ширины прямоугольника
  4. area для хранения результата - площади прямоугольника
Получите исходные данные:
  1. Выведите на экран строковый литерал "Задача 1976"
  2. Выведите на экран строковый литерал "Введите высоту прямоугольника".
  3. Инициализируйте переменную temp - получите от пользователя строку и запишите ее в temp.
  4. Инициализируйте переменную height - запишите в нее результат преобразования temp в число.
  5. Выведите на экран строковый литерал "Введите ширину прямоугольника".
  6. Получите от пользователя строку и запишите ее в temp.
  7. Инициализируйте переменную width - запишите в нее результат преобразования temp в число.
Вычислите результат и оформите ответ:
  1. Инициализируйте переменную area - умножьте переменную height на width и запишите результат в area.
  2. Выведите на экран текст по шаблону "Площадь прямоугольника со сторонами $height и $width равна $area"
Убедитесь, что для всех исходных данных получается ожидаемый результат.
