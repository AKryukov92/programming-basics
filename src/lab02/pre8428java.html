К вступлению

У вас должен получиться код, похожий на фрагмент ниже.

package lab02; import java.util.Scanner; public class Example { public static void main(String[] args) { Scanner scan = new Scanner(System.in); double aDeg; aDeg = scan.nextDouble(); double aRad; aRad = aDeg * Math.PI / 180; System.out.printf("%.4f", aRad); } }

Указание единицы измерения в имени переменной помогает ориентироваться в запутанном коде.

Иногда кроме единицы измерения полезно указывать измеряемую величину. Вот список наиболее распространенных величин:

  • Высота - height
  • Длина (предметов) - length
  • Ширина - width
  • Масса - weight
  • Объем - volume
  • Площадь - area
  • Стоимость - cost
  • Продолжительность (периодов времени) - duration
  • Количество - quantity (amount)

Главная сложность тут - расположение букв "g","h","t" в конце различных слов.

К вступлению

Вступление

В предыдущем разборе теории (открыть в новой вкладке) были рассмотрены правила оформления вычислений в программе.

Сейчас внимание будет обращено на работу с единицами измерения.

При решении прикладных задач два разных числа могут значить одно и то же. Например длина 2300 сантиметра равна длине 23 метра. Скорость 60 км/ч равна скорости 16.67 м/с.

Формула вычисления результата может быть правильной, но если в переменных будут несопоставимые единицы измерения, то результат получится некорректный.

Вычисления нужно проводить над сопоставимыми единицами измерения. Метры складывать с метрами, а километры с километрами. Для получения скорости км/ч делить километры на часы, а не метры на секунды.

Если исходные данные содержат различные единицы измерения одной и той же величины, то нужно преобразовать их все в какую-нибудь общую единицу измерения. Если даны метры, сантиметры и километры, то целесообразнее выразить каждую величину в метрах.

В задании требуется перевести градусную меру угла в радианы. В начале программы нужно получить от пользователя исходные данные.

package lab02; import java.util.Scanner; public class Example { public static void main(String[] args) { Scanner scan = new Scanner(System.in); double aDeg; aDeg = scan.nextDouble(); } }

Обратите внимание на название переменной aDeg. Оно состоит из двух частей: a и Deg. Первая часть - "логическое" название той величины, которую хранит переменная. Вторая часть - единица измерения этой величины. Сокращение от слова "Degree" - градус угла по английски.

Дополните ваш код:

  • Объявите переменную для хранения величины "a" в радианах (по английски "Radian").
  • Присвойте этой переменной результат вычисления по формуле перевода из градусов в радианы.
  • Выведите на экран значение угла в радианах с точностью до 4 знаков после запятой.
