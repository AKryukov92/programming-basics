В образцах решения задач намеренно используется очень большое количество переменных. Таким образом выделен каждый важный шаг программы. Некоторые из этих переменных нужны для решения задачи. Большинство других переменных - временные. Если они используются только в одном месте, то их полезно сократить.

Пример программы, использующей служебную временную переменную (C#).

string t = Console.ReadLine();//Считываем значение из консоли int a = int.Parse(t);//Преобразуем строковое значение в числовое

Сокращенный код (C#)

int a = int.Parse(Console.ReadLine());//Получаем целое число от пользователя

Если же переменная используется в нескольких других выражениях, то сократить ее не удастся. В этом случае ей лучше дать осмысленное имя. Более подробно об именовании временных переменных можно прочитать в главе 11.2 части III книги "Совершенный код"(открыть в новой вкладке).
