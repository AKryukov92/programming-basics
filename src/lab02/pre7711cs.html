К вступлению

У вас должен получиться код, похожий на фрагмент ниже.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String temp; temp = Console.ReadLine(); double aDeg; double aRad; aDeg = double.Parse(temp); aRad = aDeg * Math.PI / 180; double result = Math.Sin(aRad); Console.WriteLine("{0:F4}", result); } } }

Вызов метода можно писать как часть арифметического выражения. Следующий код тоже даст верный результат.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String temp; temp = Console.ReadLine(); double aDeg; aDeg = double.Parse(temp); double result = Math.Sin(aDeg * Math.PI / 180); Console.WriteLine("{0:F4}", result); } } }

Нахождение синуса угла - не единственное применение методов. Более того, мы уже применяли методы раньше.

Шаг 2/3

К шагу 1 К вступлению

Обратите внимание, что выражение "Console.ReadLine" и "double.Parse" это тоже вызовы метода. Это значит, что код программы можно переписать без потери смысла вот так:

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { double result = Math.Sin(double.Parse(Console.ReadLine()) * Math.PI / 180); Console.WriteLine("{0:F4}", result); } } }

Попробуйте выразить в одну строку программу расчета по формуле result = sin(a) + cos(a). Значение переменной "a" вводится пользователем один раз, но используется в двух местах формулы. Метод для нахождения косинуса угла называется "Math.Cos" и он тоже ожидает значение в радианах.

Шаг 3/3

К шагу 2 К вступлению

Выражение не удастся написать в одну строку, когда одно и то же значение переменной "a" используется несколько раз. Поэтому ее значение так или иначе придется записать в отдельную переменную.

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { double aRad = double.Parse(Console.ReadLine()) * Math.PI / 180; double result = Math.Sin(aRad) + Math.Cos(aRad); Console.WriteLine("{0:F4}", result); } } }

Подробный и сокращенный способ оформления для компьютера выглядят одинаково и выполняются с равной скоростью. Отличие есть только для людей, читающих код.

Вступление

В предыдущем разборе теории (открыть в новой вкладке) были рассмотрены арифметические операторы и простые операнды.

Сейчас будет разобран пример использования операнда - "Вызов метода".

Один из способов применения методов - оформление сложных вычислений: возведения в степень, нахождения синуса угла или квадратного корня числа. Есть и другие способы. О них будет рассказано в других лабораторных работах.

Различные методы применяются похожим способом. Для объяснения принципа, рассмотрим метод нахождения синуса угла. Допустим значение угла было записано в переменную "aRad". Тогда после выполнения команды:

double result = Math.Sin(aRad);

В переменную "result" будет записано значение синуса для угла "aRad".

Выражение Math.Sin это название метода. Сразу после него, в круглых скобках пишутся аргументы метода.

В качестве аргументов метода можно передавать переменные, числа и целые выражения

Обратите внимание, что метод "Math.Sin" ожидает значение в радианах, а исходные данные в задаче указаны в градусах. Поэтому прежде чем вычислять синус, нужно перевести исходные данные из градусов в радианы.

Ожидаю, что вы это делали в предыдущей задаче (открыть в новой вкладке). Предполагаю, что оно выглядит так:

using System; using System.Collections.Generic; using System.Linq; using System.Text; using System.Threading.Tasks; namespace Example { class Program { static void Main(string[] args) { String temp; temp = Console.ReadLine(); double aDeg; double aRad; aDeg = double.Parse(temp); aRad = aDeg * Math.PI / 180; Console.WriteLine("{0:F4}", aRad); } } }

К этому алгоритму нужно добавить команду вычисления синуса.
