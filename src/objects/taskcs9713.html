Задача

Переопределите стандартный метод ToString класса Range

  1. Убедитесь, что у класса Range объявлены поля для целочисленных значений "from" и "to".
  2. В классе Range создайте публичный метод ToString без аргументов и добавьте перед типом возвращаемого значения ключевое слово override. Метод должен возвращать строку.
  3. В теле метода верните строку по шаблону "[fromValue;toValue]", где вместо fromValue будет находиться значение из поля from, а вместо toValue - значение из поля to.

Проверьте результат работы метода с помощью вспомогательной программы

  1. В методе Main класса Program проекта CodeProject создайте экземпляр класса Range, from = 10, to = 60.
  2. Выведите на экран фразу "Исходный интервал" с помощью метода Write класса Console.
  3. Вызовите метод WriteLine класса Console и передайте ему экземпляр класса Range.
  4. Вызовите у экземпляра класса Range метод Shift. Первым аргументом передайте ему значение 70.
  5. Вызовите метод WriteLine класса Console и передайте ему первым аргументом шаблон "После сдвига {0}", а вторым - экземпляр класса Range.
  6. Вызовите метод Squeeze и передайте ему значение 40.
  7. Выполните конкатенацию строкового литерала "После сжатия " и экземпляра класса Range.
  8. Результат конкатенации передайте в качестве аргумента методу WriteLine класса Console.
  9. В результате запуска метод Main класса Program, в консоли должен появиться текст:
Исходный интервал [10;60] После сдвига [80;130] После сжатия [80;90]
