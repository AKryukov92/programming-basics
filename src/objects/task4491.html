Задача

Реализуйте метод для создания экземпляров класса Point со значениями по-умолчанию

  1. Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  2. В классе Point реализуйте статический метод "zero". Он не принимает аргументов и возвращает экземпляр класса Point.
  3. В теле метода объявите переменную instance типа Point и инициализируйте ее экземпляром класса Point.
  4. Инициализируйте поля "x" и "y" переменной instance значениями 0.
  5. Верните переменную instance в качестве результата работы метода.
Используйте метод в тестовом методе distanceBetweenPointsTest
  1. Запишите в переменную src результат вызова метода zero.
  2. Запишите в переменную dest результат вызова метода zero.
  3. Проверьте, что результат вызова метода distanceBetween с аргументами src и dest вернет значение 0 с точностью 0.0001.
  4. После запуска тестовых методов, все они должны быть отмечены зеленым знаком в списке методов.
