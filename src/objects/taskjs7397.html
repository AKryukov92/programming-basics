Задача

Переведите алгоритм с естественного языка на ваш язык программирования.
  1. Выведите на экран значение строкового литерала "Задача 7397"
  2. Объявите переменную today для хранения даты и времени.
  3. Инициализируйте переменную today текущей датой и временем.
    1. Создайте новый экземпляр класса Date и запишите его в переменную today. Используйте конструктор без аргументов.
  4. Выведите на экран значение переменной today в формате "dd.MM.yyyy HH:mm:ss". Для этого выполните конкатенацию значений:
    1. Строковый литерал "Текущее время: "
    2. Результат вызова функции getFullYear у переменной today.
    3. Строковый литерал "-".
    4. Прибавьте 1 к результату вызова функции getMonth у переменной today
    5. Строковый литерал "-".
    6. Результат вызова функции getDate у переменной today.
    7. Строковый литерал " ".
    8. Результат вызова функции getHours у переменной today.
    9. Строковый литерал ":".
    10. Результат вызова функции getMinutes у переменной today.
    11. Строковый литерал ":".
    12. Результат вызова функции getSeconds у переменной today.
  5. Выведите на экран время из переменной today в системе Unix-времени.
    1. Объявите переменную todayUnix для хранения unix-времени.
    2. Разделите результат вызова функции getTime у переменной today на 1000 и округлите его вниз.
    3. Округленное значение запишите в переменную todayUnix.
    4. Выведите на экран результат конкатенации строкового литерала "Текущее unix-время: " и значения переменной todayUnix.
  6. Объявите переменную tomorrow для хранения даты и времени.
  7. Запишите в переменную tomorrow значение переменной today.
  8. К значению переменной today прибавьте 1 день и результат запишите в переменную tomorrow.
    1. Прибавьте к результату вызова функции getDate у переменной tomorrow число 1.
    2. Вызовите функцию setDate у переменной tomorrow и передайте ей в качестве аргумента значение, полученное на предыдущем шаге.
    3. Результат вызова функции setDate запишите в переменную tomorrow.
  9. Выведите на экран фразу "Время через 1 день: ", а затем значение переменной tomorrow в формате "dd.MM.yyyy HH:mm:ss"
  10. Выведите на экран фразу "Unix-время через 1 день: ", а затем время из переменной tomorrow в системе Unix-времени.
  11. Запустите программу и проверьте, что у вас получился похожий результат. Дата и время должны отличаться.
