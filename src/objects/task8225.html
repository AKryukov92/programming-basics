Задача

Реализуйте статический метод для вычисления равнодействующей двух сил под произвольным углом

Для нахождения равнодействующей, нужно вычислить:
  1. В папке для исходного кода создайте класс Force.
  2. Объявите у класса Force поля для действительных значений x, y и value.
  3. В классе Library реализуйте публичный статический метод FindResultant.
  4. Он принимает в качестве аргумента два экземпляра класса Force и возвращает экземпляр класса Force.
  5. В теле метода вычислите
    1. Скалярное произведение векторов приложения сил (producta,b) как в задаче ЛР2#6924 (открыть в новой вкладке)
    2. Угол между векторами (anglea,b) как в задаче ЛР2#6924 (открыть в новой вкладке)
    3. Координаты вектора равнодействующей сил по формулам xравн=xa+xb; yравн=ya+yb
    4. Значение равнодействующей силы по формуле valueравн = valuea2+valueb2 + 2 * valuea * valueb * cos(anglea,b)
  6. Сформируйте ответ в виде экземпляра класса Force и верните его в качестве результата.
  7. Если методу был передан экземпляр класса Force с некорректными значениями полей, метод должен выбросить исключение содержащее текст ошибки из задачи.
  8. В классе TestSuite создайте юнит тесты для проверки корректности работы метода.
  9. Проверьте юнит-тестами, что метод выбрасывает ислючение при некорректных данных.
