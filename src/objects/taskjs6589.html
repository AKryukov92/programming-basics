Задача

Реализуйте функцию для решения задачи ЛР5#6589 (открыть в новой вкладке)

  • В папке для исходного кода создайте реализуйте функцию hasIntersectionInts. Она принимает 4 аргумента.
  • В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.
  • В случае передачи некорректных значений интервалов, нужно выбросить исключение.

Проверьте корректность работы функции с помощью юнит-теста

  • Создайте юнит-тест для проверки задачи и подпишите его словами "проверка пересечения интервалов. Границы заданы 4 числами".
  • В теле юнит-теста нужно 8 раз вызвать функцию hasIntersectionInts. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Проверьте корректность результатов каждого вызова функции hasIntersectionInts с помощью функции expect в сочетании с функцией toBeFalsy или toBeTruthy.
  • Проверьте что функция hasIntersectionInts выбрасывает исключение в случае с некорректными исходными данными. Воспользуйтесь функцией expect в сочетании с toThrow.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "проверка пересечения интервалов. Границы заданы 4 числами".
