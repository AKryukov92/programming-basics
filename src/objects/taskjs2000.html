Задача

Реализуйте прототип Range для решения задачи ЛР5#6589 (открыть в новой вкладке)

  • В файле для исходного кода "lab14.js" реализуйте функцию Range. Она принимает 2 аргумента.
  • В теле функции нужно присвоить значения из аргументов полям "a" "b" этого объекта.
  • Инициализируйте поле intersects прототипа Range функцией. Реализуйте эту функцию так, чтобы она возвращала результат решения задачи.

Воспользуйтесь конструктором прототипа Range для инициализации объектов и функциями прототипа для решения задачи

  • Создайте юнит-тест для проверки задачи и подпишите его словами "проверка пересечения интервалов. Границы интервалов заданы 2 экземплярами прототипа".
  • Для каждого теста из задачи создайте экземпляры прототипа Range и поместите их в переменные r1 и r2.
    • Инициализируйте переменные r1 и r2 результатом вызова функции Range. В качестве аргументов передайте значения "a" и "b" из соответствующих столбцов тестов к задаче.
    • При каждой инициализации добавьте ключевое слово "new" после оператора присваивания и перед вызовом функции Range.
  • Вызовите функцию intersects сначала у объекта r1 и передайте ей в качестве аргумента объект r2. После этого вызовите функцию у объекта r2 и передайте ей объект r1.
  • Проверьте корректность результатов каждого вызова функции intersects с помощью функции expect в сочетании с функцией toBeTruthy или toBeFalsy.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "проверка пересечения интервалов. Границы интервалов заданы 2 экземплярами прототипа".
