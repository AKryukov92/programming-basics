Задача

Реализуйте параметризованный конструктор класса Range

  1. Убедитесь, что у класса Range объявлены поля для целочисленных значений "from" и "to".
  2. В классе Range реализуйте публичный метод, который принимает в качестве аргументов 2 целых числа и называется Range (как класс). У него должно отсутствовать указание типа возвращаемого значения.
  3. В теле метода нужно инициализировать поля класса значениями, которые были переданы в качестве аргументов. Если конструктору были переданы некорректные данные, нужно выбросить ArgumentException.
  4. Перенесите инициализацию объектов из метода Init в конструктор в тестовых методах:
    1. HasIntersectionRangesTest
    2. HasIntersectionRangesAssertionWrongLeft
    3. HasIntersectionRangesAssertionWrongRight
    4. IntersectsIntsTest
    5. IntersectsIntWrongLeftTest
    6. IntersectsIntWrongRightTest
    7. IntersectsRangeTest
    8. IntersectsRangeTestWrongLeftTest
    9. IntersectsRangeTestWrongRightTest
    10. RangeInstanceComparison
    11. Task8265test1
    12. Task8265test2
    13. Task8265test3
    14. Task8265test4
    15. Task8265test5
  5. Укажите в аргументах конструктора значения 0 и 0 в тестовых методах:
    1. InitWrongTest
    2. RangeEqualsItself
    3. RangeNotEqualNull
    4. RangeNotEqualOtherClass
  6. После запуска тестовых методов, все они должны быть отмечены зеленым знаком в списке методов.
