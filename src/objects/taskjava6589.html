Задача

Реализуйте статический метод для решения задачи ЛР5#6589 (открыть в новой вкладке)

  1. В папке для исходного кода создайте класс Range.
  2. В классе Range реализуйте публичный статический метод hasIntersection.
  3. Он принимает в качестве аргумента 4 целых числа и возвращает логическое значение.
  4. В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.
  5. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение IllegalArgumentException, содержащий текст ошибки из задачи.

Проверьте корректность работы метода с помощью юнит-теста

  1. В папке для юнит-тестов создайте класс RangeSuite.
  2. В классе RangeSuite создайте юнит-тест hasIntersectionIntsTest.
  3. В теле метода hasIntersectionIntsTest вызовите метод hasIntersection.
  4. Передайте в качестве аргументов значения 2, 3, 5 и 7.
  5. Проверьте, что метод hasIntersection вернул значение false.
  6. В теле метода hasIntersectionIntsTest вызовите метод hasIntersection еще раз.
  7. Передайте в качестве аргументов значения 11, 17, 13, 19.
  8. Проверьте, что метод hasIntersection вернул значение true.
  9. После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.

Проверьте юнит-тестами, что метод выбрасывает ислючение при некорректных данных

  1. В классе RangeSuite создайте 2 юнит-теста hasIntersectionIntsWrongRedTest и hasIntersectionIntsWrongGreenTest.
  2. Они должны проверять факт возникновения исключения IllegalArgumentException.
  3. В теле метода hasIntersectionIntsWrongRedTest вызовите метод hasIntersection.
  4. Передайте ему в качестве аргументов значения 79, 73, 83, 89.
  5. В теле метода hasIntersectionIntsWrongGreenTest вызовите метод hasIntersection.
  6. Передайте ему в качестве аргументов значения 97, 101, 107, 103.
  7. После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.
