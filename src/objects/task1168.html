Задача

Реализуйте статический метод AreSame для сравнения экземпляров класса Direction.

  1. В классе Direction создайте публичный статический метод AreSame.
  2. Он должен возвращать логическое значение и принимать в качестве аргумента два экземпляра типа Point.
  3. В теле метода AreSame верните истину, если углы и расстояния у объектов эквивалентны.
  4. Нужно учесть, что угол более 360 или менее 0 равен углу в интервале от 0 до 360.
  5. В классе GeodesySuite реализуйте юнит-тест AreSameTest.
  6. Проверьте с помощью юнит-теста, что метод AreSame возвращает корректные данные во всех случаях.
