Задача

Реализуйте НЕстатический метод для решения задачи ЛР2#1058 (открыть в новой вкладке)

  1. Убедитесь, что у класса Point объявлены поля для действительных значений "x" и "y".
  2. В классе Point реализуйте публичный нестатический метод distanceTo (без ключевого слова static перед типом возвращаемого значения).
  3. Новый метод distanceTo принимает в качестве аргумента 2 действительных числа и возвращает действительное число.
  4. В теле метода верните результат решения задачи. Для его вычисления используйте поля "x" и "y" экземпляра класса Point, у которого вызвали метод, и значения из аргументов метода.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе PointSuite создайте юнит-тест distanceToValuesTest.
  2. В теле метода distanceToValuesTest объявите переменную src типа Point и инициализируйте ее экземпляром класса Point.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Инициализируйте поля "x" и "y" переменной src значениями Xa и Ya.
    2. Вызовите метод distanceTo у экземпляра класса Point из переменной src.
    3. Передайте ему в качестве аргументов значения Xb и Yb.
    4. Добавьте выражение для проверки корректность результата вызова метода distanceTo.
  4. После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
