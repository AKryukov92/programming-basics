Задача

Реализуйте статический метод для решения задачи ЛР2#6522 (открыть в новой вкладке)

  1. В папке для исходного кода создайте класс LiquidPortion.
  2. Объявите у класса LiquidPortion поля для действительных значений volume и temperature.
  3. В классе Library реализуйте публичный статический метод MixLiquids.
  4. Он принимает в качестве аргумента два экземпляра класса LiquidPortion и возвращает экземпляр класса LiquidPortion.
  5. В теле метода вычислите объем и температуру получившейся смеси, используя значения аргументов в качестве исходных данных.
  6. Сформируйте ответ в виде экземпляра класса LiquidPortion и верните его в качестве результата.
  7. Если методу был передан экземпляр класса LiquidPortion с некорректными значениями полей, метод должен выбросить исключение ArgumentException, содержащий текст ошибки из задачи.
  8. В классе TestSuite создайте юнит тесты для проверки корректности работы метода.
  9. Проверьте юнит-тестами, что метод выбрасывает ислючение при некорректных данных.
