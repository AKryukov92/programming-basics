Задача

Реализуйте статический метод AreSame для сравнения экземпляров класса Point.

  1. В классе Point создайте публичный статический метод AreSame.
  2. Он должен возвращать логическое значение и принимать в качестве аргумента два экземпляра типа Point.
  3. В теле метода AreSame верните истину, если координаты объектов равны.
  4. В классе PointsSuite реализуйте юнит-тест AreSameTest.
  5. Проверьте с помощью юнит-теста, что метод AreSame возвращает корректные данные во всех случаях.
