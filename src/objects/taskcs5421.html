Задача

Переопределите стандартный метод ToString класса Point

  1. Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y" и поле "color", которое хранит строку.
  2. В классе Point создайте публичный метод ToString без аргументов и добавьте перед типом возвращаемого значения ключевое слово override. Метод должен возвращать строку.
  3. В теле метода верните строку по шаблону "(xValue;yValue;colorValue)", где вместо xValue будет находиться значение из поля x, вместо yValue - значение из поля y, а вместо colorValue - значение из поля color.

Проверьте результат работы метода с помощью вспомогательной программы

  1. В методе Main класса Program проекта CodeProject создайте экземпляр класса Point, x = 31, y = 37, color=red
  2. Вызовите метод WriteLine класса Console и передайте ему в качестве аргумента экземпляр класса Point.
  3. Измените конструктор класса Point так, чтобы он инициализировал поле color значением "white".
  4. Создайте еще две точки p1 и p2 с координатами (11;13) и (17;19) не указывая цвет.
  5. Вызовите метод WriteLine класса Console и передайте ему первым аргументом шаблон "Расстояние между {0} и {1} составляет {2:F4}", вторым и третьим аргументом - экземпляры точек p1 и p2. Четвертым аргументом - дистанцию между точками p1 и p2.
  6. В результате запуска метод Main класса Program, в консоли должен появиться текст:
(31;37;red) Расстояние между (11;13;white) и (17;19;white) составляет 8.4853
