Задача

Пользователь вводит две даты. Первая в формате dd.MM.yyyy HH:mm:ss, вторая в unix-времени. Определить, какая из дат позже. Обработайте потенциальное исключение при распознавании даты.
Преобразование строки в дату
  1. Объявите переменную firstString и запишите в нее дату в виде строки.
  2. Объявите переменную firstDate для хранения даты и времени (тип DateTime).
  3. Вызовите статический метод ParseExact у класса DateTime. Первым аргументом передайте значение переменной firstString, вторым - строковый литерал "dd.MM.yyyy HH:mm:ss", третьим - null.
  4. Результат вызова метода ParseExact запишите в переменную firstDate.
  5. Если метод parse выбросит исключение FormatException, выведите на экран сообщение "Дата была введена в некорректном формате".
