Задача

  1. В классе Methods реализуйте публичный статический метод Task5871 для решения задачи ЛР2#5871 (открыть в новой вкладке).
  2. Метод Task5871 должен возвращать результат решения задачи для корректных значений исходных данных.
  3. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение ArgumentException, содержащий текст ошибки из задачи.
  4. Проверьте с помощью юнит-теста результат работы метода Task5871 в случаях с корректными данными.
  5. Проверьте с помощью вспомогательной программы результат работы метода Task5871 в случаях с корректными данными.
  6. В результате запуска метода Main класса Program, в консоли должен появиться текст:
0,707107 0,881127 1,000000

Проверьте юнит-тестами, что метод выбрасывает исключение при некорректных данных

  1. В классе TestSuite создайте еще 3 юнит-теста. Назовите их как Task5871assertionX, где X - цифра от 1 до 2.
  2. Добавьте ему атрибут ExpectedException. Укажите тип ожидаемого исключения как ArgumentException.
  3. В теле метода Task5871assertionX вызовите метод Task5871 класса Methods и укажите аргументы, которые приводят к выбрасыванию исключения.
  4. После запуска юнит-тестов, они должны быть отмечены зеленым знаком в списке методов.

Добавьте во вспомогательную программу обработку исключений

  1. Вызовите метод Task5871 с аргументами из каждого теста, указанного в задаче.
  2. В случае успешного выполнения метода Task5871, выведите в консоль результат с точностью до 6 знаков после запятой.
  3. В случае возникновения исключения, перехватите его и выведите в консоль сообщение об ошибке.
