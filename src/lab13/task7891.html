Реализуйте метод для решения задачи

  1. В классе Library реализуйте публичный статический метод task7891.
  2. В качестве аргумента он принимает массив целых чисел.
  3. В теле функции создайте новый массив. Для каждой пары элементов из исходного массива нужно добавить три элемента в результирующий массив. Если в исходном массиве нечетное количество элементов, функция должна выбросить исключение.
