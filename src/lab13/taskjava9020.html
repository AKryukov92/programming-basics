Задача

  • В классе Lab02 реализуйте публичный статический метод task9020. Он принимает 1 целое число, возвращает действительное число и может выбросить исключение java.lang.IllegalArgumentException.
  • В теле метода нужно вернуть результат решения задачи ЛР2#9020 (открыть в новой вкладке) используя значение аргумента в качестве исходных данных. В случае ошибки, нужно выбросить исключение IllegalArgumentException и указать при этом текст ошибки из задачи.
  • В классе Lab02Suite создайте публичный нестатический метод task9020test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task9020test нужно 4 раз вызвать метод task9020 класса Lab02. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, которые приводят к вычислению результата.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Второй - результат вызова метода task9020. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
  • Создайте 3 публичных нестатических метода, которые ничего не возвращают и не принимают аргументов. Имя выглядит как task9020assertionX, где X - цифра от 1 до 3. Пометьте их аннотацией @Test с элементом expected равным IllegalArgumentException.class.
  • В теле метода task9020assertionX нужно вызвать метод task9020 класса Lab02 и указать значения, которые приводят к выбрасыванию исключения.
  • После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.
  • В папке с кодом создайте класс Runner9020. В этом классе создайте метод main.
  • В методе main класса Runner9020 создайте массив inX. Заполните его значениями из тестов к задаче.
  • Вызовите метод task9020 класса Lab02 столько раз, сколько тестов в задаче. В каждом вызове передайте методу в качестве аргумента элемент массива inX. Поймайте потенциальное исключение IllegalArgumentException.
  • В случае успешного выполнения метода task9020, выведите в консоль результат с точностью до 4 знаков после запятой. В случае поимки исключения, выведите в консоль сообщение об ошибке.
  • В результате запуска метода main класса Runner9020, в консоли должен появиться текст.
