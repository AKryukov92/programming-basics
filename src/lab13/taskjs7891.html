Задача

Реализуйте функцию для решения задачи

  1. В файле для исходного кода реализуйте функцию task7891.
  2. В качестве аргументов она принимает массив чисел.
  3. В теле функции создайте новый массив. Для каждой пары элементов из исходного массива нужно добавить три элемента в результирующий массив. Первый и второй элемент из тройки - копии исходной пары элементов. Третий элемент из тройки - сумма этой пары элементов. Если в исходном массиве нечетное количество элементов, функция должна выбросить исключение.

Проверьте корректность работы функции с помощью юнит-тестов

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 7891 должен получиться корректный результат".
  2. Объявите переменные data, validate, actual и expected.
  3. Запишите в переменную data массив значений исходных данных для одного теста из задачи. Например 9,8, 7,6, 5,4.
  4. Создайте копию массива data с помощью функции slice.
  5. Запишите копию массива data в переменную validate.
  6. Запишите в переменную expected массив значений, которые ожидаются в результате работы метода. Например 9,8,17, 7,6,13, 5,4,9.
  7. Вызовите функцию task7891 и передайте ей переменную data в качестве аргумента.
  8. Запишите результат вызова функции в переменную actual.
  9. Проверьте, что содержимое массивов actual и expected одинаковое с помощью функции expect в сочетании с функцией toEqual.
  10. Проверьте, что содержимое массивов data и validate одинаковое с помощью функции expect в сочетании с функцией toEqual.
  11. Проверьте, что функция выбрасывает исключение при некорректных данных с помощью функции expect в сочетании с функцией toThrow.
