Задача

  • В классе Lab02 реализуйте публичный статический метод task3943. Он принимает 2 действительных числа, возвращает действительное число и может выбросить исключение java.lang.IllegalArgumentException.
  • В теле метода нужно вернуть результат решения задачи ЛР2#3943 (открыть в новой вкладке) используя значение аргумента в качестве исходных данных. В случае ошибки, нужно выбросить исключение IllegalArgumentException и указать при этом текст ошибки из задачи.
  • В классе Lab02Suite создайте публичный нестатический метод task3943test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task3943test нужно 2 раза вызвать метод task3943 класса Lab02. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, которые приводят к вычислению результата.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Второй - результат вызова метода task3943. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
  • Создайте 2 публичных нестатических метода, которые ничего не возвращают и не принимают аргументов. Имя выглядит как task3943assertionX, где X - цифра от 1 до 2. Пометьте их аннотацией @Test с элементом expected равным IllegalArgumentException.class.
  • В теле метода task3943assertionX нужно вызвать метод task3943 класса Lab02 и указать значения, которые приводят к выбрасыванию исключения.
  • После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.
  • В папке с кодом создайте класс Runner3943. В этом классе создайте метод main.
  • В методе main класса Runner3943 создайте массивы inX и inY. Заполните их значениями из тестов к задаче.
  • Вызовите метод task3943 класса Lab02 столько раз, сколько тестов в задаче. В каждом вызове передайте методу в качестве аргументов элементы массивов inX и inY с одинаковыми индексами. Поймайте потенциальное исключение IllegalArgumentException.
  • В случае успешного выполнения метода task3943, выведите в консоль результат с точностью до 6 знаков после запятой. В случае поимки исключения, выведите в консоль сообщение об ошибке.
  • В результате запуска метода main класса Runner3943, в консоли должен появиться текст.
