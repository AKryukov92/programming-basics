Задача

  1. В классе Methods реализуйте публичный статический метод task1934 для решения задачи ЛР2#1934 (открыть в новой вкладке).
  2. Метод task1934 должен возвращать результат решения задачи для корректных значений исходных данных.
  3. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение IllegalArgumentException, содержащий текст ошибки из задачи.
  4. В папке с исходным кодом создайте класс Lab13Task1934.
  5. Создайте метод main в классе Lab13Task1934 по аналогии с методами main из задач прошлых лабораторных.
  6. Создайте еще один метод в классе Lab13Task1934. Назовите его step1934. Он ничего не возвращает и не принимает аргументов.
  7. В теле метода step1934:
    1. Выведите в консоль сообщение "Введите значения A и B".
    2. Объявите переменные a и b для хранения целых чисел.
    3. Инициализируйте переменные a и b значениями, полученными из консоли.
    4. Объявите переменную result для хранения действительных чисел.
    5. Вызовите метод task1934 класса Methods и передайте ему a и b в качестве аргументов.
    6. Результат вызова метода запишите в переменную result.
    7. Выведите на экран сообщение "Результат равен " и значение переменной result с точностью 4 знака после запятой.
  8. В теле метода main:
    1. Выведите сообщение "Начало работы программы".
    2. Вызовите метод step1934.
    3. Выведите сообщение "Программа завершена успешно".
  9. В результате запуска метода main класса Lab13Task1934, в консоли должен выводиться текст из таблицы с тестовыми данными, из колонки "Исходный результат". Возможны отличия в номерах строк.
