Задача

  1. В классе Library реализуйте публичный статический метод Task9020 для решения задачи ЛР2#9020 (открыть в новой вкладке).
  2. Метод Task9020 должен возвращать результат решения задачи для корректных значений исходных данных.
  3. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение ArgumentException, содержащий текст ошибки из задачи.
  4. Проверьте с помощью юнит-теста результат работы метода Task9020 в случаях с корректными данными.

Проверьте юнит-тестами, что метод выбрасывает исключение при некорректных данных

  1. В классе TestSuite создайте еще 3 юнит-теста. Назовите их как Task9020assertionX, где X - цифра от 1 до 3.
  2. Добавьте каждому из юнит-тестов атрибут ExpectedException. Укажите тип ожидаемого исключения как ArgumentException.
  3. В теле метода Task9020assertionX вызовите метод Task9020 класса Library и укажите аргументы, которые приводят к выбрасыванию исключения.
  4. После запуска юнит-тестов, они должны быть отмечены зеленым знаком в списке методов.
