Задача

  1. В классе Methods реализуйте публичный статический метод task5871 для решения задачи ЛР2#5871 (открыть в новой вкладке).
  2. Метод task5871 должен возвращать результат решения задачи для корректных значений исходных данных.
  3. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение IllegalArgumentException, содержащий текст ошибки из задачи.
  4. Проверьте с помощью юнит-теста результат работы метода task5871 в случаях с корректными данными.

Проверьте юнит-тестами, что метод выбрасывает исключение при некорректных данных

  1. В классе TestSuite создайте еще 3 юнит-теста. Назовите их как task5871assertionX, где X - цифра от 1 до 2.
  2. Пометьте их аннотацией @Test с элементом expected равным IllegalArgumentException.class.
  3. В теле метода task5871assertionX вызовите метод task5871 класса Methods и укажите аргументы, которые приводят к выбрасыванию исключения.
  4. После запуска юнит-тестов, они должны быть отмечены зеленым знаком в списке методов.
