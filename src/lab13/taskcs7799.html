Задача

  1. В классе Methods реализуйте публичный статический метод Task7799 для решения задачи ЛР2#7799 (открыть в новой вкладке).
  2. Метод Task7799 должен возвращать результат решения задачи для корректных значений исходных данных.
  3. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение ArgumentException, содержащий текст ошибки из задачи.
  4. Проверьте с помощью юнит-теста результат работы метода Task7799 в случаях с корректными данными.

Проверьте юнит-тестами, что метод выбрасывает исключение при некорректных данных

  1. В классе TestSuite cоздайте 2 юнит-теста. Назовите их как Task7799assertionX, где X - цифра от 1 до 2.
  2. Добавьте каждому из юнит-тестов атрибут ExpectedException. Укажите тип ожидаемого исключения как ArgumentException.
  3. В теле метода Task7799assertion вызовите метод Task7799 класса Methods и укажите аргументы, которые приводят к выбрасыванию исключения.
  4. После запуска юнит-тестов, метод Task7799assertion должен быть отмечен зеленым знаком в списке методов.
