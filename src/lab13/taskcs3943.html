Задача

  1. В классе Methods реализуйте публичный статический метод Task3943 для решения задачи ЛР2#3943 (открыть в новой вкладке).
  2. Метод Task3943 должен возвращать результат решения задачи для корректных значений исходных данных.
  3. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение ArgumentException, содержащий текст ошибки из задачи.
  4. Создайте проект Lab13Task3943.
  5. В проекте Lab13Task3943 добавьте ссылку на проект CodeProject.
  6. Добавьте команду using пространства имен из проекта CodeProject.
  7. Создайте еще один метод в классе Program проекта Lab13Task3943. Назовите его Step3943. Он ничего не возвращает и не принимает аргументов.
  8. В теле метода Step1934:
    1. Выведите в консоль сообщение "Введите значения x и y".
    2. Объявите переменные x и y для хранения целых чисел.
    3. Инициализируйте переменные x и y значениями, полученными из консоли.
    4. Объявите переменную result для хранения действительных чисел.
    5. Вызовите метод Task1934 класса Methods и передайте ему x и y в качестве аргументов.
    6. Результат вызова метода запишите в переменную result.
    7. Выведите на экран сообщение "Результат равен " и значение переменной result с точностью 4 знака после запятой.
  9. В теле метода main:
    1. Выведите сообщение "Начало работы программы".
    2. Вызовите метод Step1934.
    3. Выведите сообщение "Программа завершена успешно".
  10. В результате запуска проекта Lab13Task3943, в консоли должен выводиться текст из таблицы с тестовыми данными, из колонки "Исходный результат". Возможны отличия в номерах строк.
