Задача

  1. В файле для исходного кода "lab13.js" реализуйте функцию task7799 для решения задачи ЛР2#7799 (открыть в новой вкладке).
  2. В теле функции task7799 нужно предусмотреть обработку некорректных исходных данных.
  3. Если в качестве аргументов были переданы некорректные значения, нужно создать исключение и "выбросить" его с помощью ключевого слова throw. При создании исключения, в скобках нужно указать текст ошибки.
  4. Если все аргументы корректны, нужно вычислить результат и вернуть его.
  5. В файле для юнит-тестов "lab13spec.js" проверьте с помощью юнит-теста результат работы функции task7799 в случаях с корректными данными.
  6. Проверьте с помощью вспомогательного файла Runner7799.html результат работы функции task7799 в случаях с корректными данными.
34,671920 61972,336135 0,000000 0,000000 0,000000

Проверьте юнит-тестами, что функция выбрасывает исключение при некорректных данных

  1. Чтобы убедиться, что функция выбрасывает исключение, нужно воспользоваться функцией expect в сочетании с функцией toThrow.
  2. Должна получиться всего 1 проверка возникновения исключения.
  3. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 7799 должен получиться корректный результат".

Добавьте во вспомогательный файл Runner7799.html обработку исключений

  1. Вызовите функцию task7799 с аргументами из каждого теста, указанного в задаче.
  2. В случае успешного выполнения функции task7799, выведите в консоль результат с точностью до 6 знаков после запятой.
  3. В случае возникновения исключения, нужно его перехватить и вывести в консоль сообщение об ошибке.
