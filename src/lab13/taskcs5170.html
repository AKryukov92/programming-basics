Задача

  1. В классе Methods реализуйте публичный статический метод Task5170 для решения задачи ЛР6#5170 (открыть в новой вкладке).
  2. Метод Task5170 должен возвращать результат решения задачи для корректных значений исходных данных.
  3. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение ArgumentException, содержащий текст ошибки из задачи.
  4. Проверьте с помощью юнит-теста результат работы метода Task5170 в случаях с корректными данными.

Проверьте юнит-тестами, что метод выбрасывает исключение при некорректных данных

  1. В классе TestSuite создайте юнит-тест, который называется Task5170assertion.
  2. Добавьте ему атрибут ExpectedException. Укажите тип ожидаемого исключения как ArgumentException.
  3. В теле метода Task5170assertion вызовите метод Task5170 класса Methods и укажите аргументы, которые приводят к выбрасыванию исключения.
  4. После запуска юнит-тестов, они должны быть отмечены зеленым знаком в списке методов.
