Задача

Пользователь вводит условное выражение. Например "x > 3" или "y<=-2". Выражение состоит из трех частей. В центре находится знак сравнения (один из <, <=, >, >=). Слева от него находится переменная (x, y или любая другая буква или слово). Справа находится число - ограничение, с которым проводится сравнение. Между частями выражения может быть 0 и более пробелов. Напечатать на экране словесное описание выражения.

value <=-2 Значение переменной value меньше либо равно числу -2
y < 0 Значение переменной y меньше числа 0
x < Справа от знака сравнения ожидается ограничение
>13 Слева от знака сравнения ожидается переменная
y 0 Отсутствует знак сравнения
x+7 Отсутствует знак сравнения

x < xA && x + w > xA В выражении должен быть только один знак сравнения
x > y > z В выражении должен быть только один знак сравнения
a x < 17 Неожиданное выражение "a" слева от переменной "x"
