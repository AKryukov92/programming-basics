Если писать блок else для каждой проверки исходных данных, то получится код с очень большой вложенностью. Для упрощения вложенных условий, рекомендуется использовать блоки с выходом (return). Чтобы ознакомиться с другими способами уменьшения вложенности конструкций, ознакомьтесь с главой 19.4 "Укрощение опасно глубокой вложенности" части IV книги "Совершенный код" (открыть в новой вкладке).
