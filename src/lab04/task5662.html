Задача

Даны вещественные числа a, b, c (a ≠ 0). Если a равно 0, вывести сообщение об ошибке "Уравнение не является квадратным". Найдите в прошлых работах формулу для вычисления дискриминанта, вычислите его и напишите одно из трех сообщений в зависимости от полученного значения.
  1. Если дискриминант равен 0, то у уравнения один вещественный корень.
  2. Если дискриминант меньше 0, то у уравнения нет вещественных корней.
  3. Если дискриминант больше 0, то у уравнения два вещественных корня.
Номер теста Вводимые данные Результат
a b c
1 2 3 Вещественных корней уравнения нет
11 13 19 Вещественных корней уравнения нет
3 17 5 У уравнения два вещественных корня
5 31 29 У уравнения два вещественных корня
3 6 3 Уравнение имеет один корень
1 2 1 Уравнение имеет один корень
0 7 13 Данное уравнение не является квадратным
