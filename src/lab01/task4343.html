Пользователь вводит несколько значений. Вывести данный фрагмент текста, заменив названия переменных их значениями.
User ID=$USER;Password=$PASS;Host=$HOST;Port=$PORT;Database=$DB;
Номер теста Вводимые данные Результат
user pass host port db
test pass localhost 5432 my_db User ID=test;Password=pass; Host=localhost;Port=5432;Database=my_db;

akryukov 123 localhost dev-node1 bank User ID=akryukov;Password=123; Host=localhost;Port=dev-node1;Database=bank;
