Задача

Переведите алгоритм, описанный ниже, с естественного языка на ваш язык программирования.
Объявите переменные:
  1. companyName для хранения названия компании.
  2. productName для хранения наименования товара.
  3. productCost для хранения цены на товар.
  4. messagePart для хранения фрагментов сообщения.
Получите данные от пользователя:
  1. Выведите на экран строковый литерал "Задача 1910".
  2. Выведите на экран строковый литерал "Введите название компании".
  3. Инициализируйте companyName - запишите в нее строку, полученную от пользователя.
  4. Выведите на экран строковый литерал "Введите наименование товара".
  5. Инициализируйте productName - запишите в нее строку, полученную от пользователя.
  6. Выведите на экран строковый литерал "Введите цену товара".
  7. Инициализируйте productCost - запишите в нее строку, полученную от пользователя.
Сформируйте ответ и выведите его на экран:
  1. Запишите в переменную messagePart результат конкатенации строкового литерала "Распродажа в ", переменной companyName и строкового литерала "!".
  2. Выведите на экран значение переменной messagePart.
  3. Запишите в переменную messagePart результат конкатенации строкового литерала "Супер цена на ", переменной productName, строкового литерала ". Всего ", переменной productCost и строкового литерала "!".
  4. Выведите на экран с новой строки значение переменной messagePart.
  5. Выведите на экран с новой строки значение строкового литерала "Торопитесь, предложение ограничено!".
Убедитесь, что для всех исходных данных получается ожидаемый результат.
Номер теста Вводимые данные Результат (result)
companyName productName productCost
