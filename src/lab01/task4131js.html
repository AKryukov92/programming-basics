Задача

Реализуйте код по описанию

Добавьте к проекту файл lab01dictation.html. Затем оформите следующие действия на языке программирования.
  1. Выведите в консоль значение строкового литерала "Задача 4131"
  2. Объявите переменную asdf и qaz.
  3. Запишите в переменную asdf значение строкового литерала "фыва".
  4. Запишите в переменную qaz значение переменной asdf.
  5. Выведите в консоль значение переменной asdf.
  6. Выведите в консоль значение переменной qaz.
Откройте файл lab01dictation.html в браузере, включите инструменты разработчика и проверьте, что в консоли был написан такой текст:
Задача 4131 фыва фыва
