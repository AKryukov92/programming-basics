Задача
Пользователь вводит несколько значений. Вывести данный фрагмент текста, заменив названия переменных их значениями. $ID на введенное значение id.
SELECT first_name, last_name, group FROM students WHERE id = '$ID';
Номер теста Вводимые данные (id) Результат
123456789 SELECT first_name, last_name, group FROM students WHERE student_id = '123456789';
fa93f SELECT first_name, last_name, group FROM students WHERE student_id = 'fa93f';
