5662

Задача

Реализуйте функцию для решения задачи ЛР5#5662 (открыть в новой вкладке)

  • В файле для исходного кода реализуйте функцию task5662. Она принимает 3 аргумента.
  • В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  • В файле с юнит-тестами вызовите функцию it. Она уже реализована в Jasmine. Первый ее аргумент - описание теста, например "в задаче 5662 должен получиться корректный результат". Второй аргумент - анонимная функция, содержащая код для проверки.
  • В теле юнит-теста нужно 5 раз вызвать функцию task5662. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Поскольку результат это строка, то корректность результатов каждого вызова функции task5662 нужно проверить с помощью функции expect в сочетании с функцией toEqual. Они также реализованы в Jasmine.
  • Сначала нужно вызвать функцию expect и передать ей в качестве аргумента результат вызова функции task5662.
  • В той же строке через точку нужно вызвать функцию toEqual. Единственный аргумент этой функции - ожидаемое значение.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 5662 должен получиться корректный результат".
