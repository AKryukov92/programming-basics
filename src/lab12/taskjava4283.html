Задача

Реализуйте метод для решения задачи ЛР7#4283 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task4283.
  • Он принимает в качестве аргументов 1 целое число и массив целых чисел. Метод возвращает целое число.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте публичный метод task4283test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task4283test нужно 4 раза вызвать статический метод task4283 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Второй - результат вызова метода task4283. Третий аргумент (точность) не указывается.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
