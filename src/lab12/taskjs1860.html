Задача

Реализуйте функцию для решения задачи ЛР1#1860 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task1860.
  2. В качестве аргументов ей будут передавать 1 строку.
  3. В теле функции task1860 нужно напечатать в консоль решение задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы функции с помощью вспомогательного файла

  1. В корне директории создайте файл Runner1860.html. Подключите в нем файл с кодом "src/lab12.js".
  2. В скриптах на странице Runner1860.html вызовите 3 раза функцию task1860.
  3. При каждом вызове функции task1860 в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  4. При открытии файла Runner1860.html, в консоли должен появиться текст:
Мы стремимся к знаниям Мы стремимся к цели Мы стремимся к результату
