Подготовка проекта

  • Создайте новое решение Stage2 типа "консольное приложение .NET Framework". Имя проекта укажите как CodeProject.
  • Убедитесь, что в проекте CodeProject присутствует класс Program. Он автоматически создается вместе с проектом типа "консольное приложение".
  • Добавьте в проекте CodeProject класс Methods. Он будет содержать методы для решения задач.
  • Добавьте к решению Stage2 проект типа "Проект модульного тестирования". Назовите его TestingProject.
  • Добавьте в проекте TestingProject класс TestSuite. Он будет содержать юнит-тесты для методов из класса Methods.
  • Пометьте класс TestSuite атрибутом TestClass.
  • В проекте TestingProject добавьте ссылку на проект CodeProject.
