Задача

Реализуйте метод для решения задачи ЛР7#9271 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task9271.
  • Он принимает в качестве аргумента массив действительных чисел и ничего не возвращает.
  • В теле метода нужно модифицировать массив, переданный в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте юнит-тест для проверки задачи task9271.
  • В теле юнит-теста нужно 7 раз объявить и инициализировать массивы действительных значений. Назовите их dataX, где X - номер теста.
  • Для каждого из массивов dataX нужно вызвать статический метод task9271 класса Lab07.
  • Нужно проверить с помощью статического метода assertArrayEquals класса Assert, что содержимое массива dataX изменилось корректно.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
