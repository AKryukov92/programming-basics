Задача

Реализуйте метод для решения задачи ЛР7#9271 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод task9271.
  2. Он принимает в качестве аргумента массив действительных чисел и ничего не возвращает.
  3. В теле метода модифицируйте массив, переданный в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте юнит-тест для проверки задачи task9271.
  2. Объявите переменные data и expected для хранения массивов действительных чисел.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений исходных данных для одного теста из задачи.
    2. Запишите в переменную expected массив значений, которые ожидаются в результате работы метода.
    3. Вызовите статический метод task9271 класса Methods и передайте ему переменную data в качестве аргумента.
    4. Когда метод task9271 завершит работу, содержание массива data должно измениться.
    5. Проверьте, что содержимое массивов data и expected одинаковое с помощью метода assertArrayEquals класса Assert.
  4. После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
