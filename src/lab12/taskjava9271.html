Задача

Реализуйте метод для решения задачи ЛР7#9271 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод task9271.
  2. Он принимает в качестве аргумента массив действительных чисел и ничего не возвращает.
  3. В теле метода модифицируйте массив, переданный в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте юнит-тест для проверки задачи task9271.
  2. В теле юнит-теста 7 раз объявите и инициализируйте массивы действительных значений. Назовите их dataX, где X - номер теста.
  3. Для каждого из массивов dataX вызовите статический метод task9271 класса Methods и передайте этот массив в качестве аргумента.
  4. Когда метод task9774 завершит работу, содержание массива dataX должно измениться.
  5. Проверьте с помощью статического метода assertArrayEquals класса Assert, что содержимое массива dataX изменилось корректно.
  6. После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
