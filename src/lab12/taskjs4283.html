Задача

Реализуйте функцию для решения задачи ЛР7#4283 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task4283.
  2. В качестве аргументов ей будут передавать 1 число и массив чисел.
  3. В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 4283 должен получиться корректный результат".
  2. В теле юнит-теста нужно 4 раза вызвать функцию task4283. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  3. Корректность результатов каждого вызова функции task4283 нужно проверить с помощью функции expect в сочетании с функцией toBeCloseTo. Для этой задачи достаточно точности в 4 знака после запятой.
  4. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 4283 должен получиться корректный результат".
