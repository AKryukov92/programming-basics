Задача

Реализуйте функцию для решения задачи ЛР5#4847 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task4847..
  2. В качестве аргументов ей будут передавать 3 числа.
  3. В теле функции нужно вернуть результат решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы функции с помощью юнит-теста

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 4847 должен получиться корректный результат".
  2. В теле юнит-теста нужно 2 раза вызвать функцию task4847. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  3. Поскольку результат это логическое значение, то корректность результатов каждого вызова функции task4847 нужно проверить с помощью функции expect в сочетании с функцией toBeTruthy или toBeFalsy.
  4. Сначала нужно вызвать функцию expect и передать ей в качестве аргумента результат вызова функции task4847.
  5. В той же строке через точку нужно вызвать функцию toBeTruthy или toBeFalsy. Им не нужно передавать аргументы.
  6. Если функция task4847 должна вернуть истину, то нужно применить toBeTruthy.
  7. Если функция task4847 должна вернуть ложь, то нужно применить toBeFalsy.
  8. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 4847 должен получиться корректный результат".
