Задача

Реализуйте метод для решения задачи ЛР5#4847 (открыть в новой вкладке)

  • В классе Lab05 реализуйте публичный статический метод task4847.
  • Он принимает 3 целых числа и возвращает логическое значение.
  • В теле метода нужно вернуть результат решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab05Suite создайте публичный нестатический метод task4847test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task4847test нужно 2 раза вызвать метод task4847 класса Lab05. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно указать в качестве единственного аргумента статического метода assertTrue или assertFalse класса Assert.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
