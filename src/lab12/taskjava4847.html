Задача

Реализуйте метод для решения задачи ЛР5#4847 (открыть в новой вкладке)

  • В классе Lab05 реализуйте публичный статический метод task4847.
  • Он принимает 3 целых числа и возвращает логическое значение.
  • В теле метода нужно вернуть результат решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab05Suite создайте юнит-тест для проверки задачи task4847.
  • В теле юнит-теста нужно 2 раза вызвать метод task4847 класса Lab05. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task4847 нужно проверить с помощью статического метода assertTrue или assertFalse класса Assert.
  • Единственный аргумент этих методов - результат работы, который нужно проверить.
  • Если метод task4847 должен вернуть истину, то нужно применить assertTrue.
  • Если метод task4847 должен вернуть ложь, то нужно применить assertFalse.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
