Задача

Реализуйте метод для решения задачи ЛР5#4847 (открыть в новой вкладке)

  • В классе Methods реализуйте публичный статический метод task4847.
  • Он принимает 3 целых числа и возвращает логическое значение.
  • В теле метода верните результат решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе TestSuite создайте юнит-тест для проверки задачи task4847.
  • В теле юнит-теста 2 раза вызовите метод task4847 класса Methods. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task4847 проверьте с помощью статического метода assertTrue или assertFalse класса Assert.
  • Единственный аргумент этих методов - результат работы, который нужно проверить.
  • Если метод task4847 должен вернуть истину, то примените assertTrue.
  • Если метод task4847 должен вернуть ложь, то примените assertFalse.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
