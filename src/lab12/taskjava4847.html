Задача

  • В классе Lab05 реализуйте публичный статический метод task4847. Он принимает 3 целых числа и возвращает логическое значение. В теле метода нужно напичатать в консоль решение задачи ЛР5#4847 (открыть в новой вкладке) используя значение аргумента как исходные данные.
  • В классе Lab05Suite создайте публичный нестатический метод task4847test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task4847test нужно 2 раза вызвать метод task4847 класса Lab05. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статических методов assertTrue и assertFalse класса Assert. Единственный аргумент этих методов - результат вызова метода task4847.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
