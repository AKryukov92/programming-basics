Задача

Реализуйте метод для решения задачи ЛР1#1860 (открыть в новой вкладке)

  • В классе Lab01 реализуйте публичный статический метод task1860.
  • Он принимает в качестве аргумента 1 строку.
  • В теле метода task1860 нужно напечатать в консоль решение задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы метода с помощью вспомогательной программы

  • В папке с кодом создайте класс Runner1860. В этом классе создайте метод main.
  • В методе main класса Runner1860 вызовите 3 раза метод task1860 класса Lab01.
  • При каждом вызове метода task1860 в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • В результате запуска метода main класса Runner1860, в консоли должен появиться текст:
Мы стремимся к знаниям Мы стремимся к цели Мы стремимся к результату
