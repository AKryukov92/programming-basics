Задача

Реализуйте функцию для решения задачи ЛР7#9271 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task9271.
  2. В качестве аргументов ей будут передавать массив чисел.
  3. В теле функции нужно модифицировать массив, переданный в качестве аргумента. Возвращать результат обычным способом не нужно.

Проверьте корректность работы функции с помощью юнит-теста

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 9271 должен получиться корректный результат".
  2. Объявите переменные data и expected.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений исходных данных для одного теста из задачи.
    2. Запишите в переменную expected массив значений, которые ожидаются в результате работы метода.
    3. Вызовите функцию task9271 и передайте ей переменную data в качестве аргумента.
    4. Когда функция task9271 завершит работу, содержание массива data должно измениться.
    5. Проверьте, что содержимое массивов data и expected одинаковое с помощью функции expect в сочетании с функцией toEqual.
