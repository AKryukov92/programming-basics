Реализуйте функцию для решения задачи ЛР7#9271 (открыть в новой вкладке)

  • В файле для исходного кода реализуйте функцию task9271.
  • В качестве аргументов ей будут передавать массив чисел.
  • В теле функции нужно модифицировать массив, переданный в качестве аргумента. Возвращать результат обычным способом не нужно.

Проверьте корректность работы функции с помощью юнит-теста

  • Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 9271 должен получиться корректный результат".
  • В теле юнит-теста нужно 7 раз объявить и инициализировать массивы действительных значений. Назовите их dataX, где X - номер теста.
  • Для каждого из массивов dataX нужно вызвать функцию task9271 и передать этот массив в качестве аргумента.
  • Корректность результатов каждого вызова функции task6497 нужно проверить с помощью функции expect в сочетании с функцией toEqual.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 9271 должен получиться корректный результат".
