Оформление, запуск и проверка юнит-тестов

Юнит тест нужен для проверки корректности работы программы. Он состоит из трех частей: Подготовка, выполнение, проверка. Для проверки используются статические методы класса Assert. Чтобы пользоваться ими, нужно импортировать класс Assert в начале файла. В IntellijIDEA тесты можно запускать по одному и все подряд. После запуска откроется панель с отчетом о выполнении юнит-тестов.
