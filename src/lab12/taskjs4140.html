Задача

Реализуйте функцию для решения задачи ЛР1#4140 (открыть в новой вкладке)

  • В файле для исходного кода реализуйте функцию task4140. Она принимает два аргумента.
  • В теле функции task4140 нужно напечатать в консоль решение задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью вспомогательного файла

  • В корне директории создайте файл Runner4140.html. Подключите в нем файл с кодом "src/lab12.js".
  • В скриптах на странице Runner4140.html вызовите 3 раза фукнцию task4140.
  • При каждом вызове функции task4140 в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • При открытии файла Runner4140.html, в консоли должен появиться текст:
Меня зовут Бонд, Джеймс Бонд. Меня зовут Джек Воробей, капитан Джек Воробей. Меня зовут Александр, Крюков Александр.
