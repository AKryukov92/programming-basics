Задача

Реализуйте метод для решения задачи ЛР2#2790 (открыть в новой вкладке)

  • В классе Lab02 реализуйте публичный статический метод task2790.
  • Он принимает в качестве аргумента 1 действительное число и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab02Suite создайте публичный метод task2790test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task2790test нужно 5 раз вызвать статический метод task2790 класса Lab02. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно указать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
