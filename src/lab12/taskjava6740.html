Задача

Реализуйте метод для решения задачи ЛР5#6740 (открыть в новой вкладке)

  • В классе Lab05 реализуйте публичный статический метод task6740.
  • Он принимает 2 целых числа и возвращает логическое значение.
  • В теле метода нужно напичатать в консоль решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab05Suite создайте публичный нестатический метод task6740test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task6740test нужно вызвать метод task6740 класса Lab05 столько раз, сколько потребуется для проверки задачи.
  • Результат каждого вызова нужно указать в качестве единственного аргумента статического метода assertTrue или assertFalse класса Assert.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
