Задача

  • В классе Lab05 реализуйте публичный статический метод task6740. Он принимает 2 целых числа и возвращает логическое значение. В теле метода нужно напичатать в консоль решение задачи ЛР5#6740 (открыть в новой вкладке) используя значение аргумента как исходные данные.
  • В классе Lab05Suite создайте публичный нестатический метод task6740test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task6740test нужно вызвать метод task6740 класса Lab05 столько раз, сколько потребуется для проверки задачи. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статических методов assertTrue и assertFalse класса Assert. Единственный аргумент этих методов - результат вызова метода task6740.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
