Задача

Реализуйте функцию для решения задачи ЛР7#3946 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task3946.
  2. В качестве аргументов ей будут передавать массив чисел.
  3. В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 3946 должен получиться корректный результат".
  2. Объявите переменную data.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений для одного теста из задачи.
    2. Вызовите функцию task3946 и передайте ей переменную data в качестве аргумента.
    3. Проверьте корректность результата вызова функции task3946 с помощью функции expect в сочетании с функцией toBeCloseTo.
  4. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 3946 должен получиться корректный результат".
