Задача

Реализуйте метод для решения задачи ЛР7#9774 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task9774.
  • Он принимает в качестве аргументов 1 действительное число, массив действительных чисел и ничего не возвращает.
  • В теле метода нужно модифицировать массив, переданный в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте юнит-тест для проверки задачи task9774.
  • В теле юнит-теста нужно 4 раза объявить и инициализировать массивы действительных значений. Назовите их dataX, где X - номер теста.
  • Для каждого из массивов dataX нужно вызвать статический метод task9774 класса Lab07 и передать этот массив в качестве аргумента.
  • Когда метод task9774 завершит работу, содержание массива dataX изменится.
  • При вызове метода assertArrayEquals нужно передавать массив dataX в качестве результата (вторым аргументом).
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
