Задача

Реализуйте метод для решения задачи ЛР7#9774 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task9774.
  • Он принимает в качестве аргументов 1 действительное число, массив действительных чисел и ничего не возвращает.
  • В теле метода нужно модифицировать массив, переданный в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте публичный метод task9774test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task9774test нужно 4 раза объявить и инициализировать массивы действительных значений. Назовите их dataX, где X - номер теста.
  • Для каждого из массивов dataX нужно вызвать статический метод task9774 класса Lab07.
  • После каждого вызова массив dataX нужно передать в качестве второго аргумента статического метода assertArrayEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
