Задача

Реализуйте функцию для решения задачи ЛР6#3669 (открыть в новой вкладке)

  • В файле для исходного кода реализуйте функцию task3669. Она принимает 2 аргумента.
  • В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  • В файле с юнит-тестами вызовите функцию it. Она уже реализована в Jasmine. Первый ее аргумент - описание теста, например "в задаче 3669 должен получиться корректный результат". Второй аргумент - анонимная функция, содержащая код для проверки.
  • В теле анонимной функции нужно 6 раз вызвать функцию task3669. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова функции task3669 нужно проверить с помощью функции expect в сочетании с функцией toBeCloseTo. Для этой задачи достаточно точности в 4 знака после запятой.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 3669 должен получиться корректный результат".
