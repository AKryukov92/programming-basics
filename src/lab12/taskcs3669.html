Задача

Реализуйте метод для решения задачи ЛР6#3669 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод Task3669.
  2. Он принимает в качестве аргумента 2 целых числа и возвращает большое целое число.
  3. В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте публичный метод Task3669test (далее "юнит-тест"). Добавьте ему атрибут TestMethod. Сам метод ничего не возвращает и не принимает аргументы.
  2. В теле юнит-теста 6 раз вызовите статический метод task3669 класса Methods. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  3. Корректность результатов каждого вызова метода task3669 проверьте с помощью статического метода AreEqual класса Assert.
  4. После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
