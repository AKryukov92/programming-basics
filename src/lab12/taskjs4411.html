Задача

Реализуйте функцию для решения задачи ЛР2#4411 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task4411.
  2. В качестве аргументов ей будут передавать 1 число.
  3. В теле функции нужно вернуть результат решения задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  1. В файле с юнит-тестами вызовите функцию it. Она уже реализована в Jasmine. Первый ее аргумент - описание теста, например "в задаче 4411 должен получиться корректный результат". Второй аргумент - анонимная функция, содержащая код для проверки.
  2. В теле юнит-теста объявите переменную actual.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Вызовите функцию task4411.
    2. Передайте ей в качестве аргументов значения исходных данных для одного теста из задачи.
    3. Результат вызова функции запишите в переменную actual.
    4. Поскольку результат это число, то корректность результатов каждого вызова функции task4411 нужно проверить с помощью функции expect в сочетании с функцией toBeCloseTo. Они также реализованы в Jasmine.
    5. Вызовите функцию expect и передайте ей в качестве аргумента переменную actual.
    6. В той же строке через точку вызовите функцию toBeCloseTo. Первый аргумент этой функции - ожидаемое значение. Второй - точность. Это количество знаков после запятой, которые должны совпадать в результате. Для этой задачи достаточно указать 4.
  4. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 4411 должен получиться корректный результат".
