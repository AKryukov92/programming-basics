Задача

Реализуйте функцию для решения задачи ЛР2#4411 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task4411.
  2. В качестве аргументов ей будут передавать 1 число.
  3. В теле функции нужно вернуть результат решения задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  1. В файле с юнит-тестами вызовите функцию it. Она уже реализована в Jasmine. Первый ее аргумент - описание теста, например "в задаче 4411 должен получиться корректный результат". Второй аргумент - анонимная функция, содержащая код для проверки.
  2. В теле анонимной функции нужно 3 раза вызвать функцию task4411. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  3. Поскольку результат это число, то корректность результатов каждого вызова функции task4411 нужно проверить с помощью функции expect в сочетании с функцией toBeCloseTo. Они также реализованы в Jasmine.
  4. Сначала нужно вызвать функцию expect и передать ей в качестве аргумента результат вызова функции task4411.
  5. В той же строке через точку нужно вызвать функцию toBeCloseTo. Первый аргумент этой функции - ожидаемое значение. Второй - количество знаков в дробной части, которые должны совпадать в результате. Для этой задачи достаточно указать 4.
  6. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 4411 должен получиться корректный результат".
