Задача

Реализуйте функцию для решения задачи ЛР7#1292 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task1292.
  2. В качестве аргументов ей будут передавать 1 символ.
  3. В теле функции нужно напечатать в консоль решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы функции с помощью юнит-теста

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 1292 должен получиться корректный результат".
  2. В теле юнит-теста нужно 5 раз вызвать функцию task1292 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  3. Корректность результатов каждого вызова функции task1292 нужно проверить с помощью функции expect в сочетании с функцией toEqual.
  4. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 1292 должен получиться корректный результат".
