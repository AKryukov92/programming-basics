Задача

  • В классе Lab02 реализуйте публичный статический метод task5789. Он принимает в качестве аргумента 3 действительных числа и возвращает строку. В теле метода нужно вернуть результат решения задачи ЛР2#5789 (открыть в новой вкладке) используя значения аргументов в качестве исходных данных.
  • В классе Lab02Suite создайте публичный метод task5789test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task5789test нужно 7 раз вызвать статический метод task5789 класса Lab02. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Второй - результат вызова метода task5789. Третий аргумент (точность) не указывается.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
