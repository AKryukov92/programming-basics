Задача

Реализуйте метод для решения задачи ЛР6#3669 (открыть в новой вкладке)

  • В классе Lab06 реализуйте публичный статический метод task3669.
  • Он принимает в качестве аргумента 2 целых числа и возвращает большое целое число.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab06Suite создайте публичный метод task3669test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task3669test нужно 6 раз вызвать статический метод task3669 класса Lab06. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно указать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий аргумент (точность) не указывается.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
