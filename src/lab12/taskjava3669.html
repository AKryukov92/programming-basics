Задача

Реализуйте метод для решения задачи ЛР6#3669 (открыть в новой вкладке)

  • В классе Methods реализуйте публичный статический метод task3669.
  • Он принимает в качестве аргумента 2 целых числа и возвращает большое целое число.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе TestSuite создайте публичный метод task3669test (далее "юнит-тест"). Пометьте его аннотацией @Test. Сам метод ничего не возвращает и не принимает аргументы.
  • В теле юнит-теста 6 раз вызовите статический метод task3669 класса Methods. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task3669 проверьте с помощью статического метода assertEquals класса Assert.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
