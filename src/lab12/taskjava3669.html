Задача

  • В классе Lab06 реализуйте публичный статический метод task3669. Он принимает в качестве аргумента 2 целых числа и возвращает большое целое число. В теле метода нужно вернуть результат решения задачи ЛР6#3669 (открыть в новой вкладке) используя значения аргументов в качестве исходных данных.
  • В классе Lab06Suite создайте публичный метод task3669test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task3669test нужно 5 раз вызвать статический метод task3669 класса Lab06. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Второй - результат вызова метода task3669. Третий аргумент (точность) не указывается.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
