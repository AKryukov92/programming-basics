Задача

Реализуйте метод для решения задачи ЛР7#9774 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод Task9774.
  2. Он принимает в качестве аргументов 1 действительное число, массив действительных чисел и ничего не возвращает.
  3. В теле метода модифицируйте массив, переданный в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте юнит-тест для проверки задачи Task9774.
  2. Объявите переменные data и expected для хранения массивов действительных чисел.
  3. Объявите переменную m для хранения действительного числа.
  4. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений исходных данных для одного теста из задачи.
    2. Запишите в переменную expected массив значений, которые ожидаются в результате работы метода.
    3. Запишите в переменную m значение множителя для одного теста из задачи.
    4. Вызовите статический метод Task9774 класса Methods и передайте и переменные m и data в качестве аргументов.
    5. Когда метод Task9774 завершит работу, содержание массива data должно измениться.
    6. Вызовите метод AreEqual класса CollectionAssert. Передайте переменную expected первым аргументом. Передайте переменную data вторым аргументом.
  5. После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
