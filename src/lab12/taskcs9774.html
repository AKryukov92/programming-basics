Задача

Реализуйте метод для решения задачи ЛР7#9774 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод Task9774.
  • Он принимает в качестве аргументов 1 действительное число, массив действительных чисел и ничего не возвращает.
  • В теле метода модифицируйте массив, переданный в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  • Убедитесь, что класс Lab07Suite существует.
  • В классе Lab07Suite создайте юнит-тест для проверки задачи Task9774.
  • В теле юнит-теста 4 раза объявите и инициализируйте массивы действительных значений. Назовите их dataX, где X - номер теста.
  • Для каждого из массивов dataX вызовите статический метод Task9774 класса Lab07 и передайте этот массив в качестве аргумента.
  • Когда метод Task9774 завершит работу, содержание массива dataX должно измениться.
  • Чтобы это проверить, при вызове метода AreEqual класса CollectionAssert передайте массив dataX в качестве результата (вторым аргументом).
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
