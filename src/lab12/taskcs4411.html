Задача

Реализуйте метод для решения задачи ЛР2#4411 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод Task4411.
  2. Он принимает в качестве аргумента 1 действительное число и возвращает действительное число.
  3. В теле метода верните результат решения задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы метода с помощью вспомогательной программы

  1. В методе main класса Program объявите переменную result для хранения действительных чисел.
  2. Для каждой строки тестовых данных из задачи напишите в методе main фрагмент кода:
    1. Вызовите статический метод Task4411 класса Methods.
    2. Передайте ему в качестве аргументов значения исходных данных для одного теста из задачи.
    3. Результат вызова метода запишите в переменную result.
    4. Напечатайте в консоль значение переменной result с точностью 4 знака после запятой.
  3. В результате запуска метода main класса Program, в консоли должен появиться текст:
18.1300 -7912.0000 7.0000

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте публичный метод Task4411test (далее "юнит-тест"). Он ничего не возвращает и не принимает аргументы.
  2. Добавьте методу Task4411test атрибут TestMethod.
  3. В теле юнит-теста объявите переменную actual для хранения действительных чисел.
  4. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Вызовите статический метод Task4411 класса Methods.
    2. Передайте ему в качестве аргументов значения исходных данных для одного теста из задачи.
    3. Результат вызова метода запишите в переменную actual.
    4. Вызовите статический метод AreEqual класса Assert.
    5. Первый аргумент метода AreEqual это ожидаемое значение. Второй аргумент - переменная actual. При проверке действительных чисел указывается третий аргумент - точность, равная 0.0001.
  5. Изменяйте код и перезапускайте юнит-тест пока он не будет отмечен зеленым знаком в отчете о запуске.
