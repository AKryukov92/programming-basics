Задача

Реализуйте функцию для решения задачи ЛР7#6497 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task6497.
  2. В качестве аргументов ей будут передавать 1 число.
  3. В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 6497 должен получиться корректный результат".
  2. Объявите переменные data и standard.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений исходных данных для одного теста из задачи.
    2. Запишите в переменную standard массив значений, которые ожидаются в результате работы метода.
    3. Вызовите функцию task6497 и передайте ей переменную data в качестве аргумента.
    4. Вызовите функцию expect и передайте ей в качестве аргумента массив data.
    5. В той же строке через точку вызовите функцию toEqual и передайте ей в качестве аргумента переменную standard.
  4. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 9774 должен получиться корректный результат".
