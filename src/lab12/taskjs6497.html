Реализуйте функцию для решения задачи ЛР7#6497 (открыть в новой вкладке)

  • В файле для исходного кода реализуйте функцию task6497. Она принимает 1 аргумент.
  • В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  • Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 6497 должен получиться корректный результат".
  • В теле юнит-теста нужно 4 раза вызвать функцию task6497. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова функции task6497 нужно проверить с помощью функции expect в сочетании с функцией toEqual. Для этой задачи достаточно точности в 4 знака после запятой.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 6497 должен получиться корректный результат".
