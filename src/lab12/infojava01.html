Подготовка проекта

Создайте проект с помощью maven.
  1. Откройте IntellijIDEA. Выберите пункт меню File, затем New, затем Project.
  2. В появившемся окне на панели слева выберите Maven. Нажмите Next.
  3. В поле groupId укажите "(ваш логин на гитхабе).github.io". В поле artifactId укажите "stage2". Нажмите Next.
  4. Проверьте что Project name содержит "stage2". Установите свойство Project location так, чтобы в нем был абсолютный путь к вашему репозиторию.
  5. Нажмите Finish.
У вас должен создаться проект со следующей структурой файлов:
  1. Содержимое папки с исходным кодом "src/main/java" будет компилироваться и компоноваться в запускаемую программу.
  2. Папка "src/main/resources" предназначена для хранения файлов с исходными данными.
  3. Содержимое папки для юнит-тестов "src/test/java" будет компилироваться и использоваться для проверки корректности программы.
  4. В папке с исходным кодом создайте класс Methods. Он будет содержать методы для решения задач.
  5. В папке с исходным кодом создайте класс Program. Это вспомогательная программа для проверки методов из класса Methods.
  6. Создайте метод main в классе Program по аналогии с методами main из задач прошлых лабораторных.
  7. В папке для юнит-тестов создайте класс TestSuite. Он будет содержать юнит-тесты для методов из класса Methods.
