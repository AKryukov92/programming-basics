Подготовка проекта

Создайте следующую структуру проекта:
java +-.idea +-out +-src +-Program.java +-Methods.java +-test +-TestSuite.java
  • Отметьте папку "src" как "Sources Root". Это значит, что ее содержимое будет компилироваться и компоноваться в запускаемую программу.
  • Отметьте папку test как "Test Sources Root". Это значит, что ее содержимое будет компилироваться и использоваться для проверки корректности программы.
  • Класс Methods будет содержать методы для решения задач.
  • Класс Program будет вспомогательной программой для проверки методов из класса Methods.
  • Создайте метод main в классе Program по аналогии с методами main из задач прошлых лабораторных.
  • Класс TestSuite будет содержать юнит-тесты для методов из класса Methods.
