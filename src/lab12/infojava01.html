Подготовка проекта

Создайте следующую структуру проекта:
java +-.idea +-out +-src +-solutions +-runners +-test
  • Отметьте папку "src" как "Sources Root".
  • Папка "src/solutions" предназначена для классов с методами для решения задачи.
  • Папка "src/runners" предназначена для классов со вспомогательными программами для проверки.
  • Отметьте папку test как "Test Sources Root". В этой папке будут размещаться классы с юнит-тестами.
  • В папке src создайте класс Lab01. В папке test создайте класс Lab01Suite.
