Задача

Реализуйте метод для решения задачи ЛР1#2632 (открыть в новой вкладке)

  • В классе Lab01 реализуйте публичный статический метод task2632.
  • Он принимает в качестве аргументов 3 целых числа и 1 строку.
  • В теле метода нужно напечатать в консоль решение задачи, используя значение аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью вспомогательной программы

  • В папке с кодом создайте класс Runner2632. В этом классе создайте метод main.
  • В методе main класса Runner2632 нужно 3 раза вызвать метод task2632 класса Lab01.
  • При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • В результате запуска тестового метода в консоли должен появиться текст:
<circle cx="50" cy="50" r="40" fill="yellow"/> <circle cx="400" cy="300" r="200" fill="green"/> <circle cx="254" cy="356" r="100" fill="#cc6600"/>
