Задача

Реализуйте метод для решения задачи ЛР7#6497 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод Task6497.
  2. Он принимает в качестве аргумента массив целых чисел и возвращает массив действительных чисел.
  3. В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте юнит-тест для проверки задачи Task6497.
  2. Объявите переменные data и expected для хранения массивов действительных чисел.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений исходных данных для одного теста из задачи.
    2. Запишите в переменную expected массив значений, которые ожидаются в результате работы метода.
    3. Вызовите статический метод Task6497 класса Methods и передайте переменную data в качестве аргумента.
    4. Вызовите метод AreEqual класса CollectionAssert. Передайте переменную expected первым аргументом. Передайте переменную data вторым аргументом.
  4. После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
