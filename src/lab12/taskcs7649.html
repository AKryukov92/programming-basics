Строки являются неизменными, поэтому конкатенация с помощью оператора "+" в цикле приводит к созданию новых объектов, что в свою очередь расходует драгоценную память. Для решения этой проблемы был создан класс System.Text.StringBuilder, который позволяет более эффективно работать над модификацией строки. Класс является mutable, то есть изменяемым — используйте его, если Вы хотите изменять содержимое строки. StringBuilder был введен в .NET Framework 3.5. Вам нужно с осторожностью использовать его в многопоточных средах. Больше информации: об операторе "+" для строк по русски (в новой вкладке), документация о классе StringBuilder (в новой вкладке)
