Задача

Реализуйте метод для решения задачи ЛР7#6497 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task6497.
  • Он принимает в качестве аргумента массив целых чисел и возвращает массив действительных чисел.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте публичный метод task6497test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task6497test нужно 4 раза вызвать статический метод task6497 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно передать в качестве второго аргумента статического метода assertArrayEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
