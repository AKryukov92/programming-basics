Задача

Реализуйте метод для решения задачи ЛР7#6497 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task6497.
  • Он принимает в качестве аргумента массив целых чисел и возвращает массив действительных чисел.
  • В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • Убедитесь, что класс Lab07Suite существует.
  • В классе Lab07Suite создайте юнит-тест для проверки задачи task6497.
  • В теле юнит-теста 4 раза вызовите статический метод task6497 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task6497 проверьте с помощью статического метода assertArrayEquals класса Assert.
  • Первый аргумент метода assertArrayEquals это массив ожидаемых значений. Второй аргумент - результат вызова метода task6497. При проверке массивов действительных чисел указывается третий аргумент - точность, равная 0.001.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
