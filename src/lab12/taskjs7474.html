Задача

Реализуйте функцию для решения задачи ЛР1#7474 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task7474.
  2. В качестве аргументов ей будут передавать 3 числа.
  3. В теле функции нужно напечатать в консоль решение задачи используя значение аргумента как исходные данные.

Проверьте корректность работы функции с помощью вспомогательной программы

  1. В корне директории создайте файл Runner7474.html. Подключите в нем файл с кодом "src/lab12.js".
  2. В скриптах на странице Runner7474.html нужно 2 раза вызвать функцию task7474.
  3. При каждом вызове функции task7474 в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  4. При открытии файла Runner7474.html, в консоли должен появиться текст:
