Задача

Реализуйте метод для решения задачи ЛР1#2632 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод task2632.
  2. Он принимает в качестве аргументов 3 целых числа и 1 строку.
  3. В теле метода task2632 напечатайте в консоль решение задачи, используя значение аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью вспомогательной программы

  1. В методе main класса Program 3 раза вызовите метод task2632 класса Methods.
  2. При каждом вызове метода task2632 в качестве аргументов передайте значения из разных тестов, описанных в задаче.
