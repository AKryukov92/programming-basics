Задача

  • В классе Lab07 реализуйте публичный статический метод task3946. Он принимает в качестве аргумента массив целых числа и возвращает действительное число. В теле метода нужно вернуть результат решения задачи ЛР7#3946 (открыть в новой вкладке) используя значения аргументов в качестве исходных данных.
  • В классе Lab07Suite создайте публичный метод task3946test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task3946test нужно 3 раз вызвать статический метод task3946 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно сравнить с ожидаемым результатом с помощью статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Второй - результат вызова метода task3946. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
