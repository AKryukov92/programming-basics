Задача

Реализуйте метод для решения задачи ЛР7#3946 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task3946.
  • Он принимает в качестве аргумента массив целых чисел и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте юнит-тест для проверки задачи task3946.
  • В теле юнит-теста нужно 3 раз вызвать статический метод task3946 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task3946 нужно проверить с помощью статического метода assertEquals класса Assert.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
