Задача

Реализуйте метод для решения задачи ЛР7#3946 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task3946.
  • Он принимает в качестве аргумента массив целых чисел и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте публичный метод task3946test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task3946test нужно 3 раз вызвать статический метод task3946 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно передать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
