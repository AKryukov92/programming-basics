Задача

Реализуйте метод для решения задачи ЛР5#4847 (открыть в новой вкладке)

  • В классе Lab05 реализуйте публичный статический метод Task4847.
  • Он принимает 3 целых числа и возвращает логическое значение.
  • В теле метода верните результат решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • Убедитесь, что класс Lab05Suite существует.
  • В классе Lab05Suite создайте юнит-тест для проверки задачи Task4847.
  • В теле юнит-теста 2 раза вызовите метод Task4847 класса Lab05. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода Task4847 проверьте с помощью статического метода IsTrue или IsFalse класса Assert.
  • Единственный аргумент этих методов - результат работы, который нужно проверить.
  • Если метод Task4847 должен вернуть истину, то примените IsTrue.
  • Если метод Task4847 должен вернуть ложь, то примените IsFalse.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
