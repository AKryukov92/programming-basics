Задача

Реализуйте метод для решения задачи ЛР5#5662 (открыть в новой вкладке)

  • В классе Methods реализуйте публичный статический метод task5662.
  • Он принимает в качестве аргумента 3 действительных числа и возвращает строку.
  • В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе TestSuite создайте публичный метод task5662test (далее "юнит-тест"). Пометьте его аннотацией @Test. Сам метод ничего не возвращает и не принимает аргументы.
  • В теле юнит-теста 5 раз вызовите статический метод task5662 класса Methods. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task5662 проверьте с помощью статического метода assertEquals класса Assert.
  • Первый аргумент метода assertEquals это ожидаемое значение. Второй аргумент - результат вызова метода task5662. Третий аргумент (точность) в этом случае не указывается. Она нужна только для проверки действительных чисел.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
