Задача

Реализуйте метод для решения задачи ЛР5#5662 (открыть в новой вкладке)

  • В классе Lab05 реализуйте публичный статический метод task5662.
  • Он принимает в качестве аргумента 3 действительных числа и возвращает строку.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab05Suite создайте публичный метод task5662test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task5662test нужно 5 раз вызвать статический метод task5662 класса Lab05. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно передать как второй аргумент статического метода assertEquals класса Assert. Первый аргумент - текст ожидаемого результата. Третий аргумент (точность) не указывается.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
