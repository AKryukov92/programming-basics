Задача

Реализуйте метод для решения задачи ЛР7#1292 (открыть в новой вкладке)

  • В классе Methods реализуйте публичный статический метод task1292.
  • Он принимает символ и возвращает строку.
  • В теле метода верните результат решения задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе TestSuite создайте юнит-тест для проверки задачи task1292.
  • В теле юнит-теста 5 раз вызовите метод task1292 класса Methods. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task1292 проверьте с помощью статического метода assertEquals класса Assert.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
