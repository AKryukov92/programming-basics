Задача

Реализуйте метод для решения задачи ЛР7#1292 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод task1292.
  • Он принимает символ и возвращает строку.
  • В теле метода нужно напичатать в консоль решение задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab07Suite создайте публичный нестатический метод task1292test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task1292test нужно 5 раз вызвать метод task1292 класса Lab07. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно указать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий аргумент (точность) не указывается.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
