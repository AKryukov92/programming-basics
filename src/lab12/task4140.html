Задача

Реализуйте метод для решения задачи ЛР1#4140 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод task4140.
  2. Он принимает в качестве аргументов 2 строки.
  3. В теле метода task4140 напечатайте в консоль решение задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью вспомогательной программы

  1. В методе main класса Program 3 раза вызовите метод task4140 класса Methods.
  2. При каждом вызове метода task4140 в качестве аргументов передайте значения из разных тестов, описанных в задаче.
  3. В результате запуска метода main класса Program, в консоли должен появиться текст:
Меня зовут Бонд, Джеймс Бонд. Меня зовут Джек Воробей, капитан Джек Воробей. Меня зовут Александр, Крюков Александр.
