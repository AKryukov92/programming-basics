Задача

Реализуйте метод для решения задачи ЛР1#4140 (открыть в новой вкладке)

  • В классе Lab01 реализуйте публичный статический метод task4140.
  • Он принимает в качестве аргументов 2 строки.
  • В теле метода task4140 напечатайте в консоль решение задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью вспомогательной программы

  • Для реализации вспомогательной программы создайте класс Runner4140.
  • Создайте метод main в классе Runner4140 по аналогии с методами main из задач прошлых лабораторных.
  • В методе main класса Runner4140 3 раза вызовите метод task4140 класса Lab01.
  • При каждом вызове метода task4140 в качестве аргументов передайте значения из разных тестов, описанных в задаче.
  • В результате запуска метода main класса Runner4140, в консоли должен появиться текст:
Меня зовут Бонд, Джеймс Бонд. Меня зовут Джек Воробей, капитан Джек Воробей. Меня зовут Александр, Крюков Александр.
