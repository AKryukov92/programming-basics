Задача

Реализуйте метод для решения задачи ЛР7#3946 (открыть в новой вкладке)

  1. В классе Library реализуйте публичный статический метод Task3946.
  2. Он принимает в качестве аргумента массив целых чисел и возвращает действительное число.
  3. В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте юнит-тест для проверки задачи Task3946.
  2. Объявите переменную data для хранения массива целых чисел.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений исходных данных для одного теста из задачи.
    2. Вызовите статический метод Task3946 класса Library и передайте ему переменную data в качестве аргумента.
    3. Проверьте корректность результата вызова метода Task3946 с помощью статического метода AreEqual класса Assert.
  4. После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
