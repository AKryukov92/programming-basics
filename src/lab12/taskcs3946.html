Задача

Реализуйте метод для решения задачи ЛР7#3946 (открыть в новой вкладке)

  • В классе Lab07 реализуйте публичный статический метод Task3946.
  • Он принимает в качестве аргумента массив целых чисел и возвращает действительное число.
  • В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • Создайте класс Lab07Suite для юнит-тестов. Пометьте его атрибутом TestClass.
  • В классе Lab07Suite создайте юнит-тест для проверки задачи Task3946.
  • В теле юнит-теста 3 раза вызовите статический метод Task3946 класса Lab07. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода Task3946 проверьте с помощью статического метода AreEqual класса Assert.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
