Задача

Реализуйте метод для решения задачи ЛР1#4140 (открыть в новой вкладке)

  • В классе Lab01 реализуйте публичный статический метод task4140.
  • Он принимает в качестве аргументов 2 строки.
  • В теле метода task4140 нужно напечатать в консоль решение задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью вспомогательной программы

  • В папке с кодом создайте класс Runner4140. В этом классе создайте метод main.
  • В методе main класса Runner4140 нужно 3 раза вызвать метод task4140 класса Lab01.
  • При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • В результате запуска тестового метода в консоли должен появиться текст:
Меня зовут Бонд, Джеймс Бонд. Меня зовут Джек Воробей, капитан Джек Воробей. Меня зовут Александр, Крюков Александр.
