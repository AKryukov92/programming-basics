Задача

  • В классе Lab01 реализуйте публичный статический метод task4140. Он принимает в качестве аргументов 2 строки. В теле метода нужно напечатать в консоль решение задачи ЛР1#4140 (открыть в новой вкладке) используя значения аргументов в качестве исходных данных.
  • В классе Lab01Suite создайте публичный нестатический метод task4140test. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода task4140test нужно 3 раза вызвать метод task4140 класса Lab01. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • В результате запуска тестового метода в консоли должен появиться текст:
Меня зовут Бонд, Джеймс Бонд. Меня зовут Джек Воробей, капитан Джек Воробей. Меня зовут Александр, Крюков Александр.
