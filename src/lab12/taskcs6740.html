Задача

Реализуйте метод для решения задачи ЛР5#6740 (открыть в новой вкладке)

  • В классе Lab05 реализуйте публичный статический метод Task6740.
  • Он принимает 2 целых числа и возвращает логическое значение.
  • В теле метода верните результат решения задачи, используя значение аргумента как исходные данные.

Проверьте корректность работы метода с помощью юнит-теста

  • Убедитесь, что класс Lab05Suite существует.
  • В классе Lab05Suite создайте юнит-тест для проверки задачи Task6740.
  • В теле юнит-теста вызовите метод Task6740 класса Lab05 столько раз, сколько потребуется для проверки задачи.
  • Результат каждого вызова метода Task6740 проверьте с помощью статического метода IsTrue или IsFalse класса Assert.
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
