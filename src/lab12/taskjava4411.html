Задача

Реализуйте метод для решения задачи ЛР2#4411 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод task4411.
  2. Он принимает в качестве аргумента 1 действительное число и возвращает действительное число.
  3. В теле метода верните результат решения задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте публичный метод task4411test (далее "юнит-тест"). Пометьте его аннотацией @Test. Сам метод ничего не возвращает и не принимает аргументы.
  2. В теле юнит-теста объявите переменную actual для хранения действительных чисел.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Вызовите статический метод task4411 класса Methods.
    2. Передайте ему в качестве аргументов значения исходных данных для одного теста из задачи.
    3. Результат вызова метода запишите в переменную actual.
    4. Вызовите статический метод assertEquals класса Assert.
    5. Первый аргумент метода assertEquals это ожидаемое значение. Второй аргумент - переменная actual. При проверке действительных чисел указывается третий аргумент - точность, равная 0.0001.
  4. Изменяйте код и перезапускайте юнит-тест пока он не будет отмечен зеленым знаком в отчете о запуске.
