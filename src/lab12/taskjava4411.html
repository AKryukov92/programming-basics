Задача

Реализуйте метод для решения задачи ЛР2#4411 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод task4411.
  2. Он принимает в качестве аргумента 1 действительное число и возвращает действительное число.
  3. В теле метода верните результат решения задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе TestSuite создайте публичный метод task4411test (далее "юнит-тест"). Пометьте его аннотацией @Test. Сам метод ничего не возвращает и не принимает аргументы.
  2. В теле юнит-теста 3 раза вызовите статический метод task4411 класса Methods. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  3. Корректность результатов каждого вызова метода task4411 проверьте с помощью статического метода assertEquals класса Assert.
  4. Первый аргумент метода assertEquals это ожидаемое значение. Второй аргумент - результат вызова метода task4411. При проверке действительных чисел указывается третий аргумент - точность, равная 0.001
  5. После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
