Задача

Реализуйте метод для решения задачи ЛР2#4411 (открыть в новой вкладке)

  • В классе Lab02 реализуйте публичный статический метод task4411.
  • Он принимает в качестве аргумента 1 действительное число и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи, используя значение аргумента в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе Lab02Suite создайте публичный метод task4411test (далее "юнит-тест"). Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле юнит-теста нужно 3 раза вызвать статический метод task4411 класса Lab02. Каждый раз в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова метода task4411 нужно проверить с помощью статического метода assertEquals класса Assert.
  • Первый аргумент метода assertEquals это ожидаемое значение. Второй аргумент - результат вызова метода task4411. При проверке действительных чисел указывается третий аргумент - точность, равная 0.001
  • После запуска юнит-теста, он должен быть отмечен зеленым знаком в списке методов.
