Задача

Реализуйте функцию для решения задачи ЛР7#9774 (открыть в новой вкладке)

  1. В файле для исходного кода реализуйте функцию task9774.
  2. В качестве аргументов ей будут передавать 1 число и массив чисел.
  3. В теле функции нужно модифицировать массив, переданный в качестве аргумента. Возвращать результат обычным способом не нужно.

Проверьте корректность работы функции с помощью юнит-теста

  1. Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 9774 должен получиться корректный результат".
  2. Объявите переменные m, data и expected.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Запишите в переменную data массив значений исходных данных для одного теста из задачи.
    2. Запишите в переменную expected массив значений, которые ожидаются в результате работы метода.
    3. Запишите в переменную m значение множителя для одного теста из задачи.
    4. Вызовите функцию task9774 и передайте ей переменную m и data в качестве аргументов.
    5. Когда функция task9774 завершит работу, содержание массива data должно измениться.
    6. Вызовите функцию expect и передайте ей в качестве аргумента массив data.
    7. В той же строке через точку вызовите функцию toEqual и передайте ей в качестве аргумента переменную expected.
  4. При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 9774 должен получиться корректный результат".
