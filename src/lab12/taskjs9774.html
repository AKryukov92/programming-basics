Задача

Реализуйте функцию для решения задачи ЛР7#9774 (открыть в новой вкладке)

  • В файле для исходного кода реализуйте функцию task9774.
  • В качестве аргументов ей будут передавать 1 число и массив чисел.
  • В теле функции нужно модифицировать массив, переданный в качестве аргумента. Возвращать результат обычным способом не нужно.

Проверьте корректность работы функции с помощью юнит-теста

  • Создайте юнит-тест для проверки задачи и подпишите его словами "в задаче 9774 должен получиться корректный результат".
  • В теле юнит-теста нужно 4 раза объявить и инициализировать массивы действительных значений. Назовите их dataX, где X - номер теста.
  • Для каждого из массивов dataX нужно вызвать функцию task9774 и передать этот массив в качестве аргумента.
  • Когда функция task9774 завершит работу, содержание массива dataX должно измениться.
  • Корректность результата нужно проверить с помощью функции expect в сочетании с функцией toEqual.
  • Сначала нужно вызвать функцию expect и передать ей в качестве аргумента массив dataX.
  • В той же строке через точку нужно вызвать функцию toEqual. Единственный аргумент этой функции - ожидаемый массив значений.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "в задаче 9774 должен получиться корректный результат".
