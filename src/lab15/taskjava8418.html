Реализуйте метод для решения задачи ЛР6#8418 (открыть в новой вкладке)

  1. В классе Methods реализуйте публичный статический метод task8418.
  2. В классе Methods импортируйте классы "java.util.Arrays" и "java.util.List".
  3. Он принимает в качестве аргумента 1 целое число и возвращает List целых чисел.
  4. В теле метода task8418 в случае получения некорректных значений аргументов, нужно выбросить исключение IllegalArgumentException и указать при этом текст ошибки из задачи.

Проверьте с помощью юнит-теста результат работы метода task8418 в случаях с корректными данными.

  1. В классе TestSuite создайте юнит-тест task8418test.
  2. В теле метода task8418test создайте переменные с названиями expected и actual типа List для целых чисел.
  3. Инициализируйте переменную expected результатом вызова статического метода asList класса Arrays. В качестве аргументов передайте числа, ожидаемые в первом тесте.
  4. Инициализируйте переменную actual результатом вызова метода task8418.
  5. Вызовите статический метод assertEquals класса Assert и передайте ему переменные expected и actual в качестве аргументов.
  6. Повторите эти действия для других тестов.
  7. Создайте юнит-тесты shouldNotAcceptNegativeArg и shouldNotAcceptZero. Укажите им тип ожидаемого исключения IllegalArgumentException.
  8. В теле метода shouldNotAcceptNegativeArg проверьте, что метод task8418 выбрасывает исключение в случае, когда первый аргумент - отрицательное число.
  9. В теле метода shouldNotAcceptNegativeArg проверьте, что метод task8418 выбрасывает исключение в случае, когда первый аргумент - ноль.
  10. После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.
