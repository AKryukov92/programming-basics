Реализуйте класс Comparator8258 для сортировки элементов коллекции по удаленности от числа 6.

  1. В папке для исходного кода создайте класс Comparator8258.
  2. Укажите, что он реализует абстрактный класс Comparer<int>.
  3. В классе Comparator8258 переопределите метод Compare, который принимает два целых числа a и b и возвращает целое число.
  4. Реализуйте метод Compare так, чтобы он возвращал указанное значение при данных значениях аргументов.
  5. В папке для юнит-тестов создайте класс ComparatorSuite.
  6. Создайте юнит-тест Comparator8258Test и проверьте что метод Compare возвращает ожидаемые значения.
