Задача

Переопределите стандартный метод hashCode для расчета хеша экземпляра класса Point

  • Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  • В классе Point добавьте еще одно поле "color", которое будет хранить строку.
  • Создайте публичный метод hashCode и пометьте его аннотацией @Override.
  • Он не принимает аргументов и возвращает целое число.
  • В теле метода верните значение, вычисляемое по формуле x+31y.

Проверьте, что хеши одинаковых экземпляров одинаковы

  • Добавьте юнит-тест samePointsHaveSameHashcode. В нем создайте два экземпляра класса Point с координатами (0;0) и (0;0).
  • Передайте результаты вызовов метода hashCode у экземпляров класса Point в качестве аргументов при вызове статического метода assertEquals класса Assert.
  • Проверьте, что вызов метода equals возвращает истину с помощью метода assertTrue класса Assert.
  • После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.

Проверьте, что одинаковые экземпляры точек заменяют друг друга при помещении в set

  • Добавьте юнит-тест hashSetContainsOnlyOneOfSamePoints. В нем создайте два экземпляра класса Point с координатами (0;0) и (0;0)
  • Создайте коллекцию pts типа HashSet для экземпляров класса Point.
  • Добавьте экземпляры класса Point в pts с помощью метода add.
  • Передайте результат вызова метода size() в качестве второго аргумента статического метода assertEquals. Первый аргумент - число 1.
  • Для каждого экземпляра точки, вызовите метод contains у экземпляра коллекции и передайте ему экземпляр точек в качестве аргумента.
  • Результат вызова contains с каждой из точек передайте в качестве аргумента статического метода assertTrue класса Assert.
  • После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.

  • Передайте результаты вызовов метода hashCode у экземпляров класса Point в качестве аргументов при вызове статического метода assertNotEquals класса Assert.
  • Проверьте, что вызов метода equals возвращает ложь с помощью метода assertFalse класса Assert.
