Задача

Пользователь вводит два массива данных одинакового размера. Первый массив содержит символы. Второй - числа. Для каждого элемента первого массива в результате получится отдельная строка. Количество символов в этой строке равно значению элемента из второго массива на том же индексе.
Номер теста Вводимые данные Результат
Данные сверху(chars) Данные слева(amounts)
q a z 3 2 5 qqq aa zzzzz
q w e r t y 3 3 1 7 2 6 qqq www e rrrrrrr tt yyyyyy
a s d f 9 0 7 2 aaaaaaaaa ddddddd ff
a s d f 3 3 3 Длины массивов не совпадают
z x c 1 2 3 4 Длины массивов не совпадают
1 1 1 Отсутствуют данные о символах
a a a Отсутствуют данные о количествах
