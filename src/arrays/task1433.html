Задача

Пользователь вводит данные - символы, разделенные символом ";"(точка с запятой). Вывести на экран индекс середины массива и элемент по этому индексу. Для четного количества элементов взять индекс ближе к началу массива.
Номер теста Вводимые данные Результат
q;w;e Длина массива: 3, индекс: 1, элемент: w
q;w;e;r Длина массива: 4, индекс: 1, элемент: w
q;w;e;r;t;y;u;i;o Длина массива: 9, индекс: 4, элемент: t
q;w;e;r;t;y;u;i;o;p Длина массива: 10, индекс: 4, элемент: t
q Длина массива: 1, индекс: 0, элемент: q
