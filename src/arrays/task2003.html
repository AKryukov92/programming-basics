Задача

Пользователь вводит символ массив символов с разделителем ";". Определить, присутствует ли в данном массиве символ "x".
Номер теста Вводимые данные (X) Результат
a;s;d;f Символ 'x' отсутствует
z;x;c;v Символ 'x' найден на позиции '1'
q;a;z;w;s;x;e;d;c Символ 'x' найден на позиции '5'
Символ 'x' отсутствует
x Символ 'x' найден на позиции '0'
x;x;x;x;x;x;x Символ 'x' найден на позиции '0'
