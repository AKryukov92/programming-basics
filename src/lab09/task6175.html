Задача

Исходный файл содержит целые числа, написанные в столбик без дополнительного оформления. Нужно вывести числа из файла в строчку через запятую. Перед первым и после последнего числа запятой быть не должно. Перехватить сгенерированные исключения и вывести указанное сообщение об ошибке.
Номер теста Содержимое файла Результат
открыть в новой вкладке 1, 2, 3, 4, 5
открыть в новой вкладке 0
открыть в новой вкладке
открыть в новой вкладке -1, -2, -3
открыть в новой вкладке Файл не существует
открыть в новой вкладке Не удается считать число
