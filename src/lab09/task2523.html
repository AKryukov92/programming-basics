Задача

Даны две ёмкости объемом 5 и 8 литров. Изначально они пустые. Пользователь должен выбирать действия так, чтобы в какой-нибудь из ёмкостей получилось 4 литра. Программа отображает список осмысленных действий. Полный список действий:
  1. Налить из крана в маленькую ёмкость
  2. Налить из крана в большую ёмкость
  3. Налить из маленькой ёмкости в большую
  4. Налить из большой ёмкости в маленькую
  5. Вылить из маленькой ёмкости в раковину
  6. Вылить из большой ёмкости в раковину
Бессмысленными считаются действия, которые не изменят содержимое ёмкостей:
  • Вылить из пустой ёмкости
  • Налить в полную ёмкость
Получившийся список действий должны быть пронумерован по порядку от 1 до конца с шагом 1. Пользователь вводит число - номер действия. После этого, программа должна определить, сколько воды получится в ёмкостях после выполнения действия с указанным номером. Переливание происходит "до краев". Излишки остаются в исходной ёмкости. Полученные значения нужно вывести на экран. Если в какой-нибудь емкости получается 4 литра, то вывести сообщение "головоломка решена". В ином случае, нужно сформировать следующий список действий для пользователя. В конце нужно вывести на экран количество действий, за которое пользователь решил задачу. При вводе некорректных номеров действий, количество действий увеличиваться не должно.
