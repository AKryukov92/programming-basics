Задача

Исходный файл содержит абсолютные имена файлов. Каждое имя на отдельной строчке. Нужно вывести на экран краткое имя файла. Имена файлов не должны содержать следующие символы:
  • Звездочка (*)
  • Вертикальная черта (|)
  • Обратная косая черта (\)
  • Двоеточие (:)
  • Двойные кавычки (")
  • Меньше (<)
  • Больше (>)
  • Вопросительный знак (?)
  • Косая черта (/)
