Задача

Даны вещественные числа a, b, c (a ≠ 0). Выяснить, имеет ли уравнение ax2 + bx + c = 0 вещественные корни.
Номер теста Вводимые данные Результат ()
a b c
1 2 3 Вещественных корней уравнения 1x^2 + 2x + 3 = 0 нет
3 17 5 У уравнения 3x^2 + 17x + 5 = 0 два вещественных корня
3 6 3 Уравнение 3x^2 + 6x + 3 = 0 имеет один корень
0 7 13 Сообщение об ошибке
