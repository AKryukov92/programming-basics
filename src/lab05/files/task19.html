Пользователь вводит координаты точки x и y. Программа проверяет, принадлежит ли эта точка закрашенной области. Изображение областей можно найти по адресу: ${папка с заданиями}/images/Области 1.png В исходном коде решения намеренно допущена логическая ошибка. Чтобы выполнить задание, нужно: - Подобрать исходные данные для проверки задачи. - Выяснить в каком случае программа работает неверно. - Оформить отчет о тестировании. Исправлять ошибку не нужно.
