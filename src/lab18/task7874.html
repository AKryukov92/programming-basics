Задача

Пользователь видит одно поле для ввода, кнопку с надписью "проверить" и поле для вывода результата. Браузер должен загадать случайное число от 1000 до 9999. Цель пользователя - угадать загаданное браузером число. Пользователь может вводить число в текстовое поле и нажимать кнопку проверки. При нажатии кнопки проверки, компьютер должен сравнить число от пользователя с загаданным. Если введенное число больше загаданного, то в поле для вывода должно появиться сообщение "Нужно меньше". Если введенное число меньше загаданного, то в поле для вывода должно появиться сообщение "Нужно больше". Если введенное число равно загаданному, то в поле для вывода должно появиться сообщение "Верно". Во всех остальных случаях, должно появляться сообщение "Не удается сравнить".
