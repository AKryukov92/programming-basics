Практика показывает, что предыдущая задача решается очень быстро, если предварительно написать ее словесное описание:

//Получить от пользователя значение width //Если width отрицательно, вывести сообщение об ошибке и завершить работу программы //Если width 0, то завершить работу программы //Напечатать символ "плюс" //Напечатать width-2 символов "минус" //Напечатать символ "плюс" и перенести на новую строку //Напечатать вертикальную черту //Напечатать width-2 символов "точка" //Напечатать вертикальную черту и перенести на новую строку //Напечатать символ "плюс" //Напечатать width-2 символов "минус" //Напечатать символ "плюс"

Такое словесное описание задачи называется "псевдокодом". Техника его использования подробно описана в главе 9.2 "Псевдокод для профи" части II книги "Совершенный код" (открыть в новой вкладке). Пишите псевдокод для сложных задач, которые не решаются с ходу.
