Задача

Инициализировать объект range так, чтобы при выполнении данных сценарев на экран был выведен указанный результат c точностью 0.0001.
Номер по порядку Сценарий Результат
range.a = 5; range.b = 11; range.slide(7); console.log("Интервал успешно изменен"); console.log(range.print()); Интервал успешно изменен [12;18]
range.a = 17; range.b = 29; range.slide(-4); console.log("Интервал успешно изменен"); console.log(range.print()); Интервал успешно изменен [13;25]
range.a = 37; range.b = 31; try { range.slide(2); console.log("Интервал успешно изменен"); console.log(range.print()); } catch(ex){ console.log(ex.message); } Некорректный интервал
