Задача

Реализуйте метод для одновременной инициализации полей класса Range

  • Убедитесь, что у класса Range объявлены поля для целочисленных значений "from" и "to".
  • В классе Range реализуйте метод init ("инициализировать"). Он принимает 2 целых числа и ничего не возвращает. В теле метода инициализируйте поля класса значениями из аргументов.
  • В тестовых методах hasIntersectionRangesTest, hasIntersectionRangesWrongLeftTest, hasIntersectionRangesWrongRightTest, intersectsIntsTest, intersectsIntWrongLeftTest, intersectsIntsWrongRightTest, intersectsRangeTest, intersectsRangeWrongLeftTest, intersectsRangesWrongRightTest замените инициализацию полей класса на вызов метода init.
  • Проверьте, что передача некорректных данных в метод init приводит к выбрасыванию исключения IllegalArgumentException.

Реализуйте метод для проверки эквивалентности двух экземпляров класса Range.

  • В классе Range создайте публичный метод equals и пометьте его аннотацией @Override.
  • Он должен возвращать логическое значение и принимать в качестве аргумента экземпляр типа Object.
  • Метод equals должен возвращать истину при сравнении экземпляра класса с самим собой.
  • Метод equals должен возвращать ложь и не выбрасывать исключений при сравнении экземпляра класса со значением null.
  • Метод equals должен возвращать ложь и не выбрасывать исключений при сравнении с экземпляром постороннего класса.
  • Результат метода equals должен зависеть от значения полей класса Range.
