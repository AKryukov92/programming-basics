Задача

Реализуйте сеттер для полей класса Point

Сеттером называется публичный метод, который вызывается для инициализации поля класса с модификатором доступа "private" (приватного поля).

Реализуйте сеттер для координат точки

  1. В классе Point замените модификатор доступа к полям "x" и "y". Вместо "public" укажите "private".
  2. В классе Point реализуйте публичный метод setCoordinates, который принимает 2 целых числа и ничего не возвращает.
  3. В теле метода инициализируйте поля класса значениями, которые были переданы в качестве аргументов.
  4. В тестовых методах distanceBetweenPointsTest, distanceToIntsTest, distanceToPointTest замените инициализацию полей класса на вызов метода setCoordinates.
  5. После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.

Реализуйте сеттер для цвета точки

  1. В классе Point объявите приватное поле "color".
  2. В классе Point реализуйте публичный метод setColor, который принимает строковое значение и ничего не возвращает.
  3. В теле метода setColor инициализируйте поле color значением, которое будет передано в качестве аргументов метода setColor.
