Задача

Переведите алгоритм с естественного языка на ваш язык программирования.
  1. Выведите на экран значение строкового литерала "Задача 7397"
  2. Импортируйте классы:
    1. java.text.SimpleDateFormat
    2. java.time.Instant
    3. java.time.temporal.ChronoUnit
    4. java.util.Date
  3. Объявите переменную today для хранения даты и времени (тип Date).
  4. Инициализируйте переменную today текущей датой и временем.
    1. Создайте новый экземпляр класса Date и запишите его в переменную today. Используйте конструктор без аргументов.
  5. Выведите на экран значение переменной today в формате "dd.MM.yyyy HH:mm:ss"
    1. Объявите переменную dateWithTime для хранения формата даты (тип SimpleDateFormat).
    2. Инициализируйте переменную dateWithTime новым экземпляром класса SimpleDateFormat. В качестве аргумента конструктора передайте строковый литерал "dd.MM.yyyy HH:mm:ss".
    3. Вызовите метод format у экземпляра класса SimpleDateFormat из переменной dateWithTime.
    4. Передайте ему в качестве аргумента экземпляр класса Date из переменной today.
    5. Выведите на экран строковый литерал "Текущее время: ", а затем на той же строке результат вызова метода format.
  6. Выведите на экран время из переменной today в системе Unix-времени.
    1. Объявите переменную todayUnix для хранения больших целых чисел (long).
    2. Запишите в переменную todayUnix результат вызова метода getTime у экземпляра класса Date из переменной today.
    3. Выведите на экран результат конкатенации строкового литерала "Текущее unix-время: " и значения переменной todayUnix.
  7. К значению переменной today прибавьте 1 день и результат запишите в переменную tomorrow.
    1. Объявите переменную todayInst для хранения даты и времени (тип Instant).
    2. Инициализируйте переменную todayInst результатом вызова метода toInstant у экземпляра класса Date из переменной today.
    3. Объявите переменную tomorrowInst для хранения даты и времени (тип Instant).
    4. Инициализируйте переменную tomorrowInst результатом вызова метода plus у экземпляра класса Instant из переменной todayInst.
    5. Передайте ей первым аргументом количество дней, а вторым - экземпляр DAYS перечисления ChronoUnit.
    6. Объявите переменную tomorrow для хранения даты и времени (тип Date)
    7. Инициализируйте переменную tomorrow результатом вызова статического метода from класса Date.
  8. Выведите на экран фразу "Время через 1 день: ", а затем значение переменной tomorrow в формате "dd.MM.yyyy HH:mm:ss".
  9. Выведите на экран фразу "Unix-время через 1 день: ", а затем время из переменной tomorrow в системе Unix-времени.
  10. Запустите программу и проверьте, что у вас получился похожий результат. Дата и время должны отличаться.
