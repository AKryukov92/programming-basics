Задача

Реализуйте метод для проверки эквивалентности двух экземпляров класса Point.

  • В классе Point создайте публичный метод equals и пометьте его аннотацией @Override.
  • Он должен возвращать логическое значение и принимать в качестве аргумента экземпляр типа Object.
  • В теле метода верните значение false

Проверьте, что экземпляр точки эквивалентен сам себе

  • Добавьте юнит тест pointEqualsItself. В нем создайте экземпляр класса Point.
  • Вызовите у экземпляра класса Point метод equals и передайте в качестве аргумента этот же экземпляр.
  • Проверьте результат вызова equals с помощью метода assertTrue класса Assert.
  • При запуске тестов, юнит-тест pointEqualsItself должен быть отмечен красным знаком.
  • Добавьте в тело метода equals выражение:
    if (this == o) return true;
  • Теперь pointEqualsItself при запуске должен быть отмечен зеленым знаком.

Проверьте, что эквивалентность точек зависит от значения их полей

  • Добавьте юнит-тест pointInstanceComparison. В нем создайте три экземпляра класса Point: "a1", "a2", "b1", "b2", "b3".
  • Инициализируйте поля x и y точек "a1" и "a2" значениями 10 и 15.
  • Инициализируйте поля x и y точки "b1" значениями 10 и 3, точки "b2" значениями 1 и 15, точки "b3" значениями 0 и 0.
  • Проверьте с помощью метода assertTrue, что все комбинации вызова метода equals для сравнения "a1" и "a2" возвращают истину.
  • Проверьте с помощью метода assertFalse, что все комбинации вызова метода equals для сравнения каждой из точек "b" и каждой из точек "a" возвращают ложь.
  • Должно получиться 14 сравнений.
  • При запуске тестов, юнит-тест pointInstanceComparison должен быть отмечен красным знаком.
  • Замените в теле метода equals выражение "return false" на выражение:
    Point point = (Point) o; if (x != point.x) return false; return y == point.y;
  • Теперь pointEqualsItself при запуске должен быть отмечен зеленым знаком.

Проверьте, что сравнение со значением null возвращает false и не выбрасывает исключение NullPointerException

  • Добавьте юнит-тест pointNotEqualNull. В нем создайте экземпляр класса Point.
  • Проверьте результат вызова equals с помощью метода assertFalse.
  • При запуске тестов, юнит-тест pointNotEqualNull должен быть отмечен красным знаком.
  • Добавьте в тело метода equals выражение:
    • if (o == null) return false;
  • Все тесты в классе PointSuite должны быть отмечены зеленым знаком.

Проверьте, что сравнение с экземпляром другого класса возвращает false и не выбрасывает никаких исключений

  • Добавьте юнит-тест pointNotEqualOther. В нем создайте экземпляр класса Point и экземпляр класса Range.
  • Проверьте результат вызова equals с помощью метода assertFalse
  • При запуске тестов, юнит-тест pointNotEqualOther должен быть отмечен красным знаком.
  • Добавьте в тело метода equals выражение:
    • if (o == null) return false;
  • Все тесты в классе PointSuite должны быть отмечены зеленым знаком.
