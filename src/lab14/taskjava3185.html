Задача

Реализуйте НЕстатический метод для решения задачи ЛР2#5789 (открыть в новой вкладке)

  • Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  • В классе Point реализуйте публичный нестатический метод distanceTo (без ключевого слова static перед типом возвращаемого значения).
  • Метод distanceTo принимает в качестве аргумента 1 экземпляр класса Point и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи. Для вычисления результата нужно использовать поля экземпляра, у которого вызвали метод (Xa и Ya), и поля экземпляра, который передали в качестве аргумента (Xb и Yb).

Проверьте корректность работы метода с помощью юнит-теста

  • В классе PointSuite создайте публичный метод distanceToPointTest. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода distanceToPointTest нужно создать 2 переменных типа Point и инициализировать их экземплярами класса Point.
  • Далее нужно 7 раз вызвать метод distanceTo у одного из экземпляров класса Point.
  • Перед каждым вызовом метода distanceTo нужно инициализировать поля экземпляров класса Point значениями из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно указать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
