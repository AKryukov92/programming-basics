Задача

Реализуйте НЕстатический метод для решения задачи ЛР2#5789 (открыть в новой вкладке)

  1. Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  2. В классе Point реализуйте публичный нестатический метод distanceTo (без ключевого слова static перед типом возвращаемого значения).
  3. Метод distanceTo принимает в качестве аргумента 1 экземпляр класса Point и возвращает действительное число.
  4. В теле метода верните результат решения задачи. Для его вычисления используйте поля "x" и "y" экземпляра, у которого вызвали метод, и поля "x" и "y" экземпляра, который передали в качестве аргумента.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе PointSuite создайте юнит-тест distanceToPointTest.
  2. В теле метода distanceToPointTest создайте 2 переменных типа Point: src и dest. Инициализируйте их экземплярами класса Point.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Инициализируйте поля "x" и "y" переменной src значениями Xa и Ya.
    2. Инициализируйте поля "x" и "y" переменной dest значениями Xb и Yb.
    3. Вызовите метод distanceTo у экземпляра класса Point из переменной src.
    4. Передайте ему в качестве аргументов переменную dest.
    5. Корректность результата вызова метода distanceTo проверьте с помощью статического метода assertEquals класса Assert.
  4. После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
