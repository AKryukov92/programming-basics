Задача

Исходный файл содержит список действий в формате CSV (справка в новой вкладке) с разделителем ";". Каждое действие описывается на отдельной строчке: наименование и необходимые значения. Действия описываются в следующем формате:
  • Создать. Наименование create. Значения: id нового прямоугольника, его координаты x, y, ширина и высота.
  • Переместить по горизонтали. Наименование shiftX. Значения: id, изменение Х. Ширина прямоугольника остается прежней.
  • Переместить по вертикали. Наименование shiftY. Значения: id, изменение Y. Высота прямоугольника остается прежней.
  • Изменить ширину. Наименование stretchX. Значения: id, изменение ширины. Координата верхнего-левого угла при этом не меняется.
  • Изменить высоту. Наименование stretchY. Значения: id, изменение высоты. Координата верхнего-левого угла при этом не меняется.
Нужно вывести в HTML файл: идентификатор каждого прямоугольника, количество корректных действий над ним, а также получившиеся фигуры в виде картинки. Порядок вывода информации о прямоугольниках не важен. При попытке совершить некорректное действие, прекратить обработку, вывести последний корректный результат и добавить в файл сообщение об ошибке.
