Задача

Реализуйте НЕстатический метод для решения задачи ЛР2#5789 (открыть в новой вкладке)

  • Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  • В классе Point реализуйте публичный нестатический метод distanceTo (без ключевого слова static перед типом возвращаемого значения).
  • Метод distanceTo принимает в качестве аргумента 2 целых числа и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи. Для вычисления результата нужно использовать поля класса (Xa и Ya) и значения из аргументов (Xb и Yb).

Проверьте корректность работы метода с помощью юнит-теста

  • В классе PointSuite создайте публичный метод distanceToIntsTest. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода distanceToIntsTest нужно создать переменную типа Point и инициализировать ее экземпляром класса Point.
  • Далее нужно 7 раз вызвать метод distanceTo у экземпляра класса Point.
  • Перед каждым вызовом метода distanceTo нужно инициализировать поля экземпляра класса Point значениями Xa и Ya из разных тестов, описанных в задаче. В качестве аргументов метода distanceTo нужно передавать значения Xb и Yb.
  • Результат каждого вызова нужно указать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
