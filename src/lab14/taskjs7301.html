Задача

Реализуйте прототип Point для решения задачи ЛР2#5789 (открыть в новой вкладке).

  • В файле для исходного кода "lab14.js" реализуйте функцию Point (конструктор прототипа). Она принимает 2 аргумента.
  • В теле функции нужно присвоить значения из аргументов полям "x" и "y" этого объекта.
  • Инициализируйте поле distanceTo прототипа Point функцией. Реализуйте эту функцию так, чтобы она возвращала результат решения задачи.

Воспользуйтесь конструктором прототипа Point для инициализации объектов и функциями прототипа для решения задачи

  • Создайте юнит-тест для проверки задачи и подпишите его словами "вычисление расстояния между точками. Координаты заданы 2 экземплярами прототипа".
  • Для каждого теста из задачи создайте экземпляры прототипа Point и поместите их в переменные p1 и p2.
    • Инициализируйте переменные p1 и p2 результатом вызова функции Point. В качестве аргументов передайте значения "x" и "y" из соответствующих столбцов тестов к задаче.
    • При каждой инициализации добавьте ключевое слово "new" после оператора присваивания и перед вызовом функции Point.
  • Вызовите функцию distanceTo сначала у объекта p1 и передайте ей в качестве аргумента объект p2. После этого вызовите функцию у объекта p2 и передайте ей объект p1.
  • Проверьте корректность результатов каждого вызова функции distanceTo c помощью функции expect в сочетании с функцией toBeCloseTo. Достаточная точность 4 знака после запятой.
  • Проверьте, что при создании экземпляра прототипа выбрасывается исключение при создании некорректных значениях аргументов. Воспользуйтесь функцией expect в сочетании с функцией toThrow.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "вычисление расстояния между точками. Координаты заданы 2 экземплярами прототипа".
