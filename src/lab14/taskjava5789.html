Задача

Реализуйте статический метод для решения задачи ЛР2#5789 (открыть в новой вкладке)

  • В папке для исходного кода создайте класс Point.
  • В классе Point реализуйте публичный статический метод distanceBetween.
  • Он принимает в качестве аргумента 4 целых числа и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  • В папке для юнит-тестов создайте класс PointSuite.
  • В классе PointSuite создайте публичный метод distanceBetweenIntsTest. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода distanceBetweenIntsTest нужно 5 раз вызвать статический метод distanceBetween класса Point. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно указать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
