Задача

Реализуйте функцию для решения задачи ЛР2#5789 (открыть в новой вкладке) внутри объекта

  • Создайте юнит-тест для проверки задачи и подпишите его словами "вычисление расстояния от одной точки до другой. Координаты другой точки заданы 1 объектом".
  • В теле юнит-теста инициализируйте объект с полями "x" и "y". Значения для полей взять из 1 и 2 столбика исходных данных первого теста.
  • Дополнительно, у юнит-теста нужно инициализировать поле distanceTo. В нем должна храниться анонимная функция, которая принимает 1 аргумент.
  • В теле функции distanceTo нужно вернуть результат решения задачи. В качестве исходных данных нужно использовать поля "x" и "y" объектов. Первая точка - этот же объект. Вторая передается в качестве аргумента функции.
  • В теле юнит-теста инициализируйте второй объект с полями "x" и "y". Значения для полей взять из 3 и 4 столбика исходных данных первого теста.
  • Вызовите функцию distanceTo у первого объекта и передайте в качестве аргумента второй объект.
  • Результат вызова функции нужно проверить с помощью функции expect в сочетании с функцией toBeCloseTo. Достаточная точность 4 знака после запятой.
  • Вызовите функцию distanceTo еще 4 раза. По одному разу для каждого оставшегося теста. Перед каждым вызовом нужно присвоить полям обоих объектов значения из соответствующего теста.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "вычисление расстояния от одной точки до другой. Координаты другой точки заданы 1 объектом".
