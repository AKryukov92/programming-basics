Задача

Реализуйте параметризованный конструктор класса Point

Конструктор класса - специальный метод, тело которого выполняется при создании экземпляра класса.
  1. Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y" и поле "color", которое хранит строку.
  2. В классе Point реализуйте конструктор. Это публичный метод, который принимает в качестве аргументов 2 целых числа и называется Point (как класс). У него должно отсутствовать указание типа возвращаемого значения.
  3. В теле метода инициализируйте поля "x" и "y" значениями, которые были переданы в качестве аргументов. Поля "color" инициализируйте значением "white".
  4. Перенесите инициализацию объектов из метода setCoordinates в конструктор в тестовых методах:
    1. distanceBetweenPointsTest
    2. distanceToIntsTest
    3. distanceToPointTest
    4. pointInstanceComparison
  5. Укажите в аргументах конструктора значения 0 и 0 в тестовых методах:
    1. pointEqualsItself
    2. pointNotEqualNull
    3. pointNotEqualOther
  6. После запуска тестовых методов, все они должны быть отмечены зеленым знаком в списке методов.
