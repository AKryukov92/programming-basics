Задача

Реализуйте параметризованный конструктор класса Point

Конструктор класса - специальный метод, тело которого выполняется при создании экземпляра класса.
  1. Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  2. В классе Point реализуйте конструктор. Это публичный метод, который принимает в качестве аргументов 2 целых числа ("px" и "py") и называется Point (как класс). У него должно отсутствовать указание типа возвращаемого значения.
  3. В теле конструктора инициализируйте поля "x" и "y" значениями, которые были переданы в качестве аргументов.
Используйте конструктор в тестовом методе distanceBetweenPointsTest
  1. Вызовите конструктор и передайте ему в качестве аргументов значения -1 и -3.
  2. Результат вызова конструктора запишите в переменную src.
  3. Вызовите конструктор и передайте ему в качестве аргументов значения 2 и 9.
  4. Результат вызова конструктора запишите в переменную dest.
  5. Проверьте, что результат вызова метода distanceBetween с аргументами src и dest вернет значение 12.3693 с точностью 0.0001.
  6. После запуска тестовых методов, все они должны быть отмечены зеленым знаком в списке методов.
