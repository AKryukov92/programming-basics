Задача

Реализуйте параметризованный конструктор класса Point

  • Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y'.
  • В классе Point реализуйте публичный метод, который принимает в качестве аргументов 2 целых числа и называется Point (как класс). У него должно отсутствовать указание типа возвращаемого значения.
  • В теле метода нужно инициализировать поля класса значениями, которые были переданы в качестве аргументов.
  • В тестовых методах distanceBetweenPointsTest, distanceToIntsTest, distanceToPointTest, pointInstanceComparison перенесите инициализацию объектов из метода setCoordinates в конструктор.
  • В тестовых методах pointEqualsItself, pointNotEqualNull, pointNotEqualOther укажите в аргументах конструктора значения 0 и 0.
  • После запуска тестовых методов, все они должны быть отмечены зеленым знаком в списке методов.
