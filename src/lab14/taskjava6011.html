Задача

Реализуйте статический метод для решения задачи ЛР2#5789 (открыть в новой вкладке) используя экземпляры класса

  • Объявите у класса Point поля для целочисленных значений "x" и "y".
  • В классе Point реализуйте еще один публичный статический метод distanceBetween.
  • Новый метод distanceBetween принимает в качестве аргумента 2 экземпляра класса Point и возвращает действительное число.
  • В теле метода нужно вернуть результат решения задачи. В вычислениях нужно использовать поля "x" и "y" экземпляров класса Point, которые были переданы в качестве аргументов.

Проверьте корректность работы метода с помощью юнит-теста

  • В классе PointSuite создайте публичный метод distanceBetweenPointsTest. Он помечен аннотацией @Test, ничего не возвращает и не принимает аргументы.
  • В теле метода distanceBetweenPointsTest нужно создать 2 переменных типа Point и инициализировать их экземплярами класса Point.
  • Далее нужно 7 раз вызвать статический метод distanceBetween класса Point.
  • Перед каждым вызовом метода distanceBetween нужно инициализировать поля "x" и "y" экземпляров класса Point значениями из разных тестов, описанных в задаче.
  • Результат каждого вызова нужно указать в качестве второго аргумента статического метода assertEquals класса Assert. Первый аргумент - ожидаемое значение. Третий - точность, равная 0.001.
  • После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
