Задача

Переопределите стандартный метод toString класса Range

  1. Убедитесь, что у класса Range объявлены поля для целочисленных значений "from" и "to".
  2. В классе Range создайте публичный метод toString без аргументов и пометьте его аннотацией @Override. Метод должен возвращать строку.
  3. В теле метода верните строку по шаблону "[fromValue;toValue]", где вместо fromValue будет находиться значение из поля from, а вместо toValue - значение из поля to.

Проверьте результат работы метода с помощью вспомогательной программы

  1. В методе Main класса Runner создайте экземпляр класса Range, from = 10, to = 60.
  2. Выведите на экран фразу "Исходный интервал" с помощью метода print.
  3. Вызовите метод println и передайте ему экземпляр класса Range.
  4. Вызовите у экземпляра класса Range метод shift. Первым аргументом передайте ему значение 70.
  5. Вызовите метод printf и передайте ему первым аргументом шаблон "После сдвига %s\n", а вторым - экземпляр класса Range.
  6. Вызовите метод squeeze и передайте ему значение 40.
  7. Вызовите метод printf. Первый аргумент "После сжатия %s". Второй аргумент - экземпляр класса Range.
  8. В результате запуска метод main класса Runner, в консоли должен появиться текст:
Исходный интервал [10;60] После сдвига [80;130] После сжатия [80;90]
