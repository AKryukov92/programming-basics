Задача

Переопределите стандартный метод toString класса Range

  • Убедитесь, что у класса Range объявлены поля для целочисленных значений "from" и "to".
  • В классе Range создайте публичный метод toString без аргументов и пометьте его аннотацией @Override. Метод должен возвращать строку.
  • В теле метода верните строку по шаблону "[fromValue;toValue]", где вместо fromValue будет находиться значение из поля from, а вместо toValue - значение из поля to.

Проверьте результат работы метода с помощью вспомогательной программы

  • В папке с кодом создайте класс Runner9713. В этом классе создайте метод main.
  • В методе main класса Runner9713 создайте экземпляр класса Range, from = 10, to = 60.
  • Выведите на экран фразу "Исходный интервал" с помощью метода print.
  • Вызовите метод println и передайте ему экземпляр класса Range.
  • Вызовите у экземпляра класса Range метод shift. Первым аргументом передайте ему значение 70.
  • Вызовите метод printf и передайте ему шаблон "После сдвига %s\n", вторым аргументом - экземпляр класса Range.
  • Вызовите метод squeeze и передайте ему значение 40.
  • Вызовите метод printf. Первый аргумент "После сжатия %s". Второй аргумент - экземпляр класса Range.
  • В результате запуска метод main класса Runner9713, в консоли должен появиться текст:
Исходный интервал [10;60] После сдвига [80;130] После сжатия [80;90]
