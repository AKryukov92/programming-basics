Задача

Реализуйте метод для создания экземпляров класса Point с указанием значений

  1. Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  2. В классе Point реализуйте статический метод "make". Он принимает в качестве аргументов два числа ("px" и "py") и возвращает экземпляр класса Point.
  3. В теле метода объявите переменную instance типа Point и инициализируйте ее экземпляром класса Point.
  4. Инициализируйте поле "x" переменной instance значением аргумента px.
  5. Инициализируйте поле "y" переменной instance значением аргумента py.
  6. Верните переменную instance в качестве результата работы метода.
Используйте метод в тестовом методе distanceBetweenPointsTest
  1. Запишите в переменную src результат вызова метода zero.
  2. Вызовите метод make и передайте ему в качестве аргумента значения -5 и -7.
  3. Результат вызова метода запишите в переменную dest.
  4. Проверьте, что результат вызова метода distanceBetween с аргументами src и dest вернет значение 8.6023 с точностью 0.0001.
  5. После запуска тестовых методов, все они должны быть отмечены зеленым знаком в списке методов.
