Задача

Реализуйте функцию для решения задачи ЛР2#5789 (открыть в новой вкладке) используя объекты

  • В файле для исходного кода "lab14.js" реализуйте функцию distanceBetweenObjects. Она принимает 2 аргумента.
  • В теле функции нужно вернуть результат решения задачи. В качестве исходных данных нужно использовать поля "x" и "y" объектов, которые были переданы в качестве аргументов.

Проверьте корректность работы метода с помощью юнит-теста

  • Создайте юнит-тест для проверки задачи и подпишите его словами "вычисление расстояния между точками. Координаты заданы 2 объектами".
  • В теле юнит-теста нужно 5 раз вызвать функцию distanceBetweenObjects.
  • Перед каждым вызовом функции distanceBetweenObjects нужно объявить два объекта и инициализировать их поля "x" и "y" значениями из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова функции distanceBetweenObjects нужно проверить с помощью функции expect в сочетании с функцией toBeCloseTo. Достаточная точность 4 знака после запятой.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "вычисление расстояния между точками. Координаты заданы 2 объектами".
