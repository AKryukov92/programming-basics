Задача

Исходный файл содержит список действий в формате CSV (справка в новой вкладке) с разделителем ";". Каждое действие описывается на отдельной строчке: наименование и необходимые значения. Действия описываются в следующем формате:
  • Создать. Наименование create. Значения: id новой фигуры, ее тип и остальные параметры. Для круга это координаты центра и радиус. Для прямоугольника: координаты верхнего-левого угла, ширина и высота.
  • Переместить по горизонтали. Наименование shiftX. Значения: id, изменение Х.
  • Переместить по вертикали. Наименование shiftY. Значения: id, изменение Y.
  • Изменить ширину. Наименование stretchX. Значения: id, изменение ширины.
  • Изменить высоту. Наименование stretchY. Значения: id, изменение высоты.
Нужно вывести на экран: идентификатор каждой фигуры, количество корректных действий над ней, а также получившиеся координаты и размеры. Порядок вывода информации о фигурах не важен. При попытке совершить некорректное действие, проигнорировать его и продолжить обработку файла. Вывести количество некорректных действий.
