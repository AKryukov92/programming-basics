Задача

Реализуйте метод для одновременной инициализации полей x и y класса Point

  • Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  • В классе Point реализуйте публичный метод setCoordinates, который принимает 2 целых числа и ничего не возвращает.
  • В теле метода инициализируйте поля класса значениями, которые были переданы в качестве аргументов.
  • В тестовых методах distanceBetweenPointsTest, distanceToIntsTest, distanceToPointTest замените инициализацию полей класса на вызов метода setCoordinates.
  • После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.
