Задача

Реализуйте функцию для решения задачи ЛР5#6589 (открыть в новой вкладке) как поле объекты

  • Создайте юнит-тест для проверки задачи и подпишите его словами "проверка пересечения одного интервала с другим. Границы другого интервала заданы 2 числами".
  • Инициализируйте объект с полями "a" и "b" в теле юнит-теста. Значения полей возьмите из 1 и 2 столбца исходных данных первого теста.
  • Инициализируйте у объекта поле intersects анонимной функцией, которая принимает 2 аргумента.
  • Реализуйте функцию intersects так, чтобы она возвращала результат решения задачи. В качестве исходных данных используйте поля "a" и "b" этого же объекта и значения из аргументов.
  • Вызовите функцию intersects у объекта и передайте ей значения из 3 и 4 столбца исходных данных первого теста.
  • Проверьте корректность результатов каждого вызова функции intersects с помощью функции expect в сочетании с функцией toBeFalsy или toBeTruthy.
  • Проверьте что функция intersects выбрасывает исключение в случае с некорректными исходными данными. Воспользуйтесь функцией expect в сочетании с toThrow.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "проверка пересечения интервалов. Границы заданы 2 объектами".
