Задача

Реализуйте статический метод для решения задачи ЛР2#5789 (открыть в новой вкладке) используя экземпляры класса

  1. Объявите у класса Point поля для действительных значений "x" и "y".
  2. В классе Point реализуйте еще один публичный статический метод distanceBetween.
  3. Новый метод distanceBetween принимает в качестве аргументов 2 экземпляра класса Point и возвращает действительное число.
  4. В теле метода верните результат решения задачи. Для его вычисления используйте поля "x" и "y" экземпляров класса Point, которые были переданы в качестве аргументов.

Проверьте корректность работы метода с помощью юнит-теста

  1. В классе PointSuite создайте юнит-тест distanceBetweenPointsTest.
  2. В теле метода distanceBetweenPointsTest объявите переменные src и dest типа Point. Инициализируйте их экземплярами класса Point.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Инициализируйте поля "x" и "y" переменной src значениями из Xa, Ya.
    2. Инициализируйте поля "x" и "y" переменной dest значениями из Xb, Yb.
    3. Вызовите статический метод distanceBetween класса Point и передайте ему в качестве аргументов значения переменных src и dest.
    4. Добавьте выражение для проверки результата вызова метода distanceBetween.
  4. После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
