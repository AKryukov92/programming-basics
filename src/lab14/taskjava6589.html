Задача

Реализуйте статический метод для решения задачи ЛР5#6589 (открыть в новой вкладке)

  1. В папке для исходного кода создайте класс Range.
  2. В классе Range реализуйте публичный статический метод hasIntersection.
  3. Он принимает в качестве аргумента 4 целых числа и возвращает логическое значение.
  4. В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.
  5. Если методу были переданы некорректные значения исходных данных, он должен выбросить исключение IllegalArgumentException, содержащий текст ошибки из задачи.

Проверьте корректность работы метода с помощью юнит-теста

  1. В папке для юнит-тестов создайте класс RangeSuite.
  2. В классе RangeSuite создайте юнит-тест hasIntersectionIntsTest.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Вызовите статический метод hasIntersection класса Range.
    2. Передайте ему в качестве аргументов значения leftA, leftB, rightA, rightB.
    3. Корректность результата вызова метода hasIntersection проверьте с помощью статических методов assertTrue или assertFalse класса Assert.
  4. После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.

Проверьте юнит-тестами, что метод выбрасывает ислючение при некорректных данных

  1. В классе RangeSuite создайте 2 юнит-теста hasIntersectionIntsWrongLeftTest и hasIntersectionIntsWrongRightTest.
  2. Они должны проверять факт возникновения исключения IllegalArgumentException.
  3. В теле метода hasIntersectionIntsWrongLeftTest вызовите метод hasIntersection и укажите такие значения, в которых левый интервал будет некорректным.
  4. В теле метода hasIntersectionIntsWrongRightTest вызовите метод hasIntersection и укажите такие значения, в которых правый интервал будет некорректным.
  5. После запуска тестовых методов, они должны быть отмечены зеленым знаком в списке методов.
