Задача

Переопределите стандартный метод toString класса Range

  • Убедитесь, что у класса Range объявлены поля для целочисленных значений "from" и "to".
  • В классе Range создайте публичный метод ToString без аргументов и добавьте перед типом возвращаемого значения ключевое слово override. Метод должен возвращать строку.
  • В теле метода верните строку по шаблону "[fromValue;toValue]", где вместо fromValue будет находиться значение из поля from, а вместо toValue - значение из поля to.

Проверьте результат работы метода с помощью вспомогательной программы

  • В методе Main класса Runner создайте экземпляр класса Range, from = 10, to = 60.
  • Выведите на экран фразу "Исходный интервал" с помощью метода print.
  • Вызовите метод WriteLine класса Console и передайте ему экземпляр класса Range.
  • Вызовите у экземпляра класса Range метод shift. Первым аргументом передайте ему значение 70.
  • Вызовите метод WriteLine класса Console и передайте ему первым аргументом шаблон "После сдвига {0}\n", а вторым - экземпляр класса Range.
  • Вызовите метод squeeze и передайте ему значение 40.
  • Вызовите метод WriteLine класса Console. Первый аргумент "После сжатия {0}". Второй аргумент - экземпляр класса Range.
  • В результате запуска метод Main класса Runner, в консоли должен появиться текст:
Исходный интервал [10;60] После сдвига [80;130] После сжатия [80;90]
