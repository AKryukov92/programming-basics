Задача

Переведите алгоритм с естественного языка на ваш язык программирования.
  1. Выведите на экран значение строкового литерала "Задача 7397"
  2. Объявите переменную today для хранения даты и времени (тип DateTime).
  3. Инициализируйте переменную today текущей датой и временем с помощью статического свойства Now класса DateTime.
  4. Выведите на экран значение переменной today в формате "dd.MM.yyyy HH:mm:ss"
    1. Вызовите метод ToString у экземпляра класса DateTime из переменной today.
    2. Передайте ему в качестве аргумента строковый литерал "dd.MM.yyyy HH:mm:ss".
    3. Выведите на экран строковый литерал "Текущее время: ", а затем на той же строке результат вызова метода ToString.
  5. Выведите на экран время из переменной today в системе Unix-времени.
    1. Объявите переменную epochStart для хранения даты и времени.
    2. Инициализируйте переменную epochStart новым экземпляром класса DateTime.
    3. В конструктор DateTime передайте первым аргументом число 1970, вторым - число 1 и третьим - число 1.
    4. Объявите переменную fromStartToNow для хранения временных интервалов (тип TimeSpan).
    5. Вычтите из переменной today значение переменной epochStart.
    6. Запишите результат вычитания в переменную fromStartToNow.
    7. Объявите переменную todayUnix для хранения больших целых чисел (long).
    8. Инициализируйте переменную todayUnix значением свойства TotalSeconds из переменной fromStartToNow. Выполните явное приведение типа.
    9. Выведите на экран результат конкатенации строкового литерала "Текущее unix-время: " и значения переменной todayUnix.
  6. Объявите переменную tomorrow для хранения даты и времени
  7. К значению переменной today прибавьте 1 день и результат запишите в переменную tomorrow.
    1. Вызовите метод AddHours у экземпляра класса DateTime из переменной today.
    2. Передайте ему в качестве аргумента количество дней.
    3. Запишите в переменную tomorrow результат вызова метода AddHours.
  8. Выведите на экран фразу "Время через 1 день: ", а затем значение переменной tomorrow в формате "dd.MM.yyyy HH:mm:ss".
  9. Выведите на экран фразу "Unix-время через 1 день: ", а затем время из переменной tomorrow в системе Unix-времени.
  10. Запустите программу и проверьте, что у вас получился похожий результат. Дата и время должны отличаться.
