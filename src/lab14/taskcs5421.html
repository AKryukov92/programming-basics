Задача

Переопределите стандартный метод ToString класса Point

  • Убедитесь, что у класса Point объявлены поля для целочисленных значений "x" и "y".
  • В классе Point создайте публичный метод ToString без аргументов и добавьте перед типом возвращаемого значения ключевое слово override. Метод должен возвращать строку
  • В теле метода верните строку по шаблону "(xValue;yValue)", где вместо xValue будет находиться значение из поля x, а вместо yValue - значение из поля y.

Проверьте результат работы метода с помощью вспомогательной программы

  • В методе Main класса Runner создайте экземпляр класса Point, x = 31, y = 37
  • Вызовите метод WriteLine класса Console и передайте ему в качестве аргумента экземпляр класса Point.
  • Создайте еще две точки p1 и p2 с координатами 11;13 и 17;19.
  • Вызовите метод WriteLine класса Console и передайте ему первым аргументом шаблон "Расстояние между {0} и {1} составляет {2:4F}\n", вторым и третьим аргументом - экземпляры точек p1 и p2. Четвертым аргументом - дистанцию между точками p1 и p2.
  • В результате запуска метод Main класса Runner, в консоли должен появиться текст:
(31;37) Расстояние между (11;13) и (17;19) составляет 8.4853
