Задача

Реализуйте функцию для решения задачи ЛР5#6589 (открыть в новой вкладке) используя объекты

  • В файле для исходного кода "lab14.js" реализуйте функцию hasIntersectionObjects. Она принимает 2 аргумента.
  • В теле функции нужно вернуть результат решения задачи. В качестве исходных данных нужно использовать поля "a" и "b" объектов, которые были переданы в качестве аргументов.

Проверьте корректность работы функции с помощью юнит-теста

  • Создайте юнит-тест для проверки задачи и подпишите его словами "проверка пересечения интервалов. Границы заданы 2 объектами".
  • В теле юнит-теста нужно 8 раз вызвать функцию hasIntersectionObjects.
  • Перед каждым вызовом функции hasIntersectionObjects нужно объявить два объекта и инициализировать их поля "a" и "b" значениями из разных тестов, описанных в задаче.
  • Проверьте корректность результатов каждого вызова функции hasIntersectionObjects с помощью функции expect в сочетании с функцией toBeFalsy или toBeTruthy.
  • Проверьте что функция hasIntersectionInts выбрасывает исключение в случае с некорректными исходными данными. Воспользуйтесь функцией expect в сочетании с toThrow.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "проверка пересечения интервалов. Границы заданы 2 объектами".
