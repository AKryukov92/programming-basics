Программа состоит из следующих частей:
* Подготовка к работе (общее оформление)
* Объявление и инициализация переменных для работы
* Получение данных
* Проверка данных
* Выполнение действий для решения задачи
* Вывод результата
* Завершение работы

Guard-выражения - условная конструкция, которая проверяет: может ли программа начинать работать.
Пишется в виде:
 if (условие) {
	 Сообщение об ошибке
	 return;
 }
