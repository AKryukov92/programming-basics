Задача

Строковый литерал. Строковые литералы в C. Это последовательность символов, которая начинается с двойной кавычки и заканчивается двойной кавычкой. Используется для оформления текста при печати на экран или других манипуляциях.
Чтобы напечатать строковый литерал содержащий фразу "Ура, заработало" в консоль, используйте команду:
System.out.println("Ура! заработало");
Выведите на экран текст
Ура! заработало! Вторая строка, вторая команда Выделенный текст можно копировать сочетанием клавиш Ctrl+C Скопированный текст можно вставить в код сочетанием клавиш Ctrl+V
Если не поставить кавычку в конце строкового литерала, то компьютер не сможет понять когда он заканчивается. Поэтому при попытке запуска компьютер выдаст три ошибки компиляции:
System.out.println("Ура! Заработало!);
  1. unclosed string literal
  2. ';' expected
  3. reached end of file while parsing

Задача

Выведите на экран текст.
{ "x":15, "y": 28.7 }
Компьютер определяет начало и конец сообщения по двойным кавычкам ". Из за того, что кавычки одинаковы, любая первая будет значить начало литерала, а следующая - конец литерала. Если будет третья - она будет значить начало нового строкового литерала. Поэтому для вывода на экран одного символа " нужно написать комбинацию символов \". Это один из спецсимволов.
Спецсимволы - комбинации символов, которые начинаются с обратного слеша и обозначают какой-то особенный символ.
Если вы не написали комбинацию символов \" там где нужно напечатать кавычку, то ошибки будут такими:
System.out.println("{ "x":15, "y": 28.7 }");
  1. ')' expected
  2. not a statement
  3. ';' expected
Нужно оформить по аналогии:
System.out.println("одинокая кавычка \" на экране");

  1. Выведите на экран значение строкового литерала "Задача 1640"
  2. Объявите переменную undersong для хранения повторяющейся строки.
  3. Инициализируйте переменную undersong - запишите в нее строковый литерал "Йо-хо-хо, и бутылка рому!".
  4. Выведите на экран результат конкатенации строкового литерала "Пятнадцать человек на сундук мертвеца." и переменной undersong.
  5. Выведите на экран результат конкатенации строкового литерала "Пей, и дьявол тебя доведет до конца." и переменной undersong.
Объявление переменной это команда из двух слов, после которых нужно поставить символ ;.
String undersong;
  1. Первое слово - тип переменной. Поначалу это будет String. В следующих практических вы познакомитесь с другими типами переменных и даже сможете создавать свои собственные типы.
  2. Второе слово - имя переменной. Вы можете самостоятельно придумывать имя переменной.
Правила именования переменных:
  1. Может состоять из русских и латинских букв, цифр, знака подчеркивания "_"
  2. Может начинаться с буквы или знака подчеркивания. С цифры начинаться не может.
  3. Не может содержать знаки математических операций: + - = / * % и прочих
  4. Название должно отражать назначение переменной
