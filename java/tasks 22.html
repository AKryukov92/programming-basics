Задача

Дана длина ребра куба. Найти площадь его боковой поверхности по формуле area=6*edge2.
  1. Объявите в классе Library публичный статический метод task9354.
  2. Он принимает в качестве аргумента действительное число (edge).
  3. Метод возвращает действительное число в качестве результата.
  4. Если значение аргумента "edge" меньше или равно 0, то метод должен выбросить исключение.
Для того, чтобы выбросить исключение, нужно добавить в тело метода следующий код:
if (edge <= 0) { throw new Exception("Значение edge должно быть положительным"); }
Когда метод может выбросить исключение, то в его сигнатуре нужно добавлять выражение throws Exception. Сигнатура этого метода выглядит так:
public static double task9354(double edge) throws Exception
В методе main класса Program добавьте код для проверки:
try { double e = 7; double res = Library.task9354(e);//Если эту команду написать за пределами блока try-catch, то возникнет ошибка компиляции: // java: unreported exception java.lang.Exception; must be caught or declared to be thrown System.out.printf("Площадь поверхности куба равна %.4f", res); res = Library.task9354(7247); System.out.printf("Площадь поверхности куба равна %.4f", res); res = Library.task9354(-11); System.out.printf("Площадь поверхности куба равна %.4f", res); } catch (Exception e) { System.out.println(e.getMessage()); } try { double res = Library.task9354(0); System.out.printf("Площадь поверхности куба равна %.4f", res); } catch (Exception e) { System.out.println(e.getMessage()); }
В результате запуска метода main, в консоли должен появиться следующий текст:
Площадь поверхности куба равна 294,0000 Площадь поверхности куба равна 315114054,0000 Значение edge должно быть положительным Значение edge должно быть положительным
