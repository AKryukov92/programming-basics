Задача

Пользователь вводит слово (goal). Нужно напечатать на экран фразу "Мы стремимся к " и затем слово от пользователя.
import java.util.Scanner;//Это нужно написать над фразой public class Scanner input = new Scanner(System.in); //создать Scanner String goal; //Объявить переменную для хранения текста goal = input.nextLine(); //Получить данные от пользователя и записать их в переменную System.out.println("Мы стремимся к " + goal); //Напечатать текст на экран
Чтобы проверить эту программу, её нужно запустить несколько раз. При каждом запуске нужно вводить только одно слово из колонки Исходные данные. Задача решена правильно, если для всех исходных данных у вас получились соответствующие результаты.
Номер тестаИсходные данные (goal)Результат
1знаниямМы стремимся к знаниям
2целиМы стремимся к цели
3результатуМы стремимся к результату
Одна из распространенных ошибок - не объявить переменную и дать команду на запись значения в переменную.
Scanner input = new Scanner(System.in); //для иллюстрации команда объявления пропущена String goal; goal = input.nextLine(); System.out.println("Мы стремимся к " + goal);
При попытке запустить такой код, компьютер выдаст ошибку: "cannot find symbol variable goal"
Другая распространенная ошибка - пропустить инициализацию переменной. То есть объявить переменную и сразу использовать её в команде печати на экран.
Scanner input = new Scanner(System.in); String goal; //для иллюстрации, команда записи значения пропущена: goal = input.nextLine(); System.out.println("Мы стремимся к " + goal);
В таком случае компьютер выдаст ошибку: "variable goal might not have been initialized"

  1. Объявите переменную companyName для хранения названия компании.
  2. Объявите переменную productName для хранения наименования товара.
  3. Объявите переменную productCost для хранения цены на товар.
  4. Объявите переменную messagePart для хранения фрагментов сообщения.
Получите данные от пользователя
  1. Выведите на экран строковый литерал "Задача 1910".
  2. Выведите на экран строковый литерал "Введите название компании".
  3. Инициализируйте companyName - запишите в нее строку, полученную от пользователя.
  4. Выведите на экран строковый литерал "Введите наименование товара".
  5. Инициализируйте productName - запишите в нее строку, полученную от пользователя.
  6. Выведите на экран строковый литерал "Введите цену товара".
  7. Инициализируйте productCost - запишите в нее строку, полученную от пользователя.
Сформируйте ответ и выведите его на экран
  1. Выполните конкатенацию строкового литерала "Распродажа в ", переменной companyName и строкового литерала "!".
  2. Запишите в переменную messagePart результат конкатенации.
  3. Выведите на экран значение переменной messagePart.
  4. Выполните конкатенацию строкового литерала "наши " и переменной productName.
  5. Запишите в переменную messagePart результат конкатенации.
  6. Выполните конкатенацию переменной messagePart и строкового литерала " по супер цене. Всего ".
  7. Запишите в переменную messagePart результат конкатенации.
  8. Запишите в переменную messagePart результат конкатенации messagePart и переменной productCost.
  9. Запишите в переменную messagePart результат конкатенации messagePart и строкового литерала "!".
  10. Выведите на экран с новой строки значение переменной messagePart.
  11. Выведите на экран с новой строки значение строкового литерала "Торопитесь, предложение ограничено!".
Убедитесь, что для всех исходных данных получается ожидаемый результат.

Задача

Пользователь вводит фамилию и имя. Нужно оформить фразу по шаблону:
Меня зовут $lastName, $firstName $lastName.
Выражения вида $lastName это условное обозначение для вас. Вместо этого выражения нужно написать значение переменной lastName. В Java это можно сделать с помощью команды System.out.printf(). В скобочках нужно написать:
  1. Шаблон текста со специальными символами %s в тех местах, где нужно написать переменную.
  2. Переменные через запятую. В том порядке, как они должны быть написаны в сообщении.
Команда для решения этой задачи выглядит так
System.out.printf("Меня зовут %s, %s %s", lastName, firstName, lastName);
