SVG (от англ. Scalable Vector Graphics — масштабируемая векторная графика) — язык разметки двумерной векторной графики в формате XML.

Особенности SVG:

  • SVG рисунок это текстовый файл с расширением .svg
  • SVG рисунки могут встраиваться в HTML страницы (с расширением .html)
  • SVG широко используется для описания векторной графики в Интернете
  • SVG описывает рисунок в формате XML.
  • Рисунки в формате SVG не теряют качество при масштабировании, растяжении или сжатии.
  • Все элементы SVG рисунка могут быть анимированы
  • SVG соответствует международным стандартам DOM и XSL

Существует множество программ для их редактирования, например Inkscape, CorelDRAW, Adobe Illustrator. В рамках данной лабораторной работы мы будем создавать векторные рисунки с помощью функций текстового вывода в файл.

Для того, чтобы немедленно увидеть результат наших действий, мы воспользуемся возможностью встраивать SVG рисунки внутрь HTML страниц.

Создайте текстовый файл, скопируйте в него следующий код.

<!DOCTYPE html> <html> <head> <meta charset="utf-8"/> <head/> <body> <h1>Мой первый векторный рисунок</h1> <svg width="800" height="600"> <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" /> </svg> </body> </html>

Сохраните файл с расширением «.html». Откройте получившийся файл в браузере. В результате вы должны увидеть следующее изображение.

Обратите внимание:

  • Описание SVG рисунка начинается с элемента <svg>
  • Высота и ширина SVG рисунка описывается атрибутами элемента <svg>. Высоте соответствует height. Ширине соответствует width.
  • Элемент <circle> используется для создания круга.
  • Атрибуты cx и cy описывают координаты центра круга по оси X и Y. Если они не будут указаны, то центр будет находиться в точке (0,0)
  • Атрибут r описывает радиус круга.
  • Атрибут stroke указывает на цвет обводки круга, а stroke-width – на толщину линии. Данный круг обведен зеленой (green) линией толщиной 4 пикселя.
  • Атрибут fill описывает цвет заливки круга. В данном примере он желтый (yellow).
  • Описание SVG рисунка заканчивается элементом </svg>

Формат SVG также поддерживает следующие фигуры:
  • Прямоугольники
  • Линии
  • а также некоторые другие
Прямоугольник описывается элементом <rect>.
  • Положение прямоугольника определяется координатами его верхнего левого угла и описывается атрибутами x и y.
  • Ширина и высота прямоугольника зависит от атрибутов width и height соответственно.
  • Цвет заливки, цвет обводки и толщина обводки описывается, так же как и для круга.
  • У всех фигур есть также свойство opacity. Оно отвечает за прозрачность фигуры и представляет собой дробное число от 0 до 1.
Линия описывается элементом <line>
  • Положение линии описывается двумя точками: x1, y1 и x2, y2.
  • Цвет линии описывается атрибутом stroke. Толщина линии описывается атрибутом stroke-width.
