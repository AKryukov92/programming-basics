Пользователь вводит число total. Выведите на экран в столбик все целые числа от 1 до total включительно. В начале программы должно появиться слово "Введение", а после всех чисел - слово "Заключение". Перед каждым числом нужно написать слово "Глава".

Конечная цель решения - компактная запись с помощью конструкции цикла. Чтобы к ней прийти, нужно сделать несколько промежуточных шагов. Первым делом нужно реализовать подробное решение для нескольких частных случаев. Нужно написать несколько условий. Первое должно работать для всех чисел. Следующее для всех чисел кроме первого. Шестое - для всех чисел, кроме первых шести. Подробное решение этой задачи выглядит так:
int total = int.Parse(Console.ReadLine()); Console.WriteLine("Введение"); if (1 <= total) { Console.WriteLine("Глава " + 1);//Это действие будет работать когда пользователь вводит числа 1,2,3,4,5,6 } if (2 <= total) { Console.WriteLine("Глава " + 2);//Это действие будет работать когда пользователь вводит числа 2,3,4,5,6 } if (3 <= total) { Console.WriteLine("Глава " + 3);//Это действие будет работать когда пользователь вводит числа 3,4,5,6 } if (4 <= total) { Console.WriteLine("Глава " + 4);//Это действие будет работать когда пользователь вводит числа 4,5,6 } if (5 <= total) { Console.WriteLine("Глава " + 5);//Это действие будет работать когда пользователь вводит числа 5,6 } if (6 <= total) { Console.WriteLine("Глава " + 6);//Это действие будет работать когда пользователь вводит число 6 } Console.WriteLine("Заключение");
Подробное решение работает корректно почти для всех тестов. В полученном результате нужно найти набор повторяющихся действий. В данном случае это:
if (1 <= total) { Console.WriteLine("Глава " + 1); }
Обратите внимания, что эти действия повторяются с небольшими изменениями. В них используются разные числа: 1, 2, 3 и так далее. Нужно это число заменить на переменную. Таким образом действия будут полностью идентичными. Код должен получиться таким:
int total = int.Parse(Console.ReadLine()); int i = 1; Console.WriteLine("Введение"); if (i <= total) { Console.WriteLine("Глава " + i); i = i + 1;//До этой команды значение переменной i равно 1. После неё оно станет равно 2. } if (i <= total) {//Значение i равно 2 Console.WriteLine("Глава " + i); i = i + 1; } if (i <= total) {//Значение i равно 3 Console.WriteLine("Глава " + i); i = i + 1; } if (i <= total) {//Значение i равно 4 Console.WriteLine("Глава " + i); i = i + 1; } if (i <= total) {//Значение i равно 5 Console.WriteLine("Глава " + i); i = i + 1; } if (i <= total) {//Значение i равно 6 Console.WriteLine("Глава " + i); i = i + 1; } Console.WriteLine("Заключение");
Как только вы написали код, который состоит из полностью одинаковых фрагментов, вы можете применить конструкцию while для автоматического повторения этих действий.
int total = int.Parse(Console.ReadLine()); int i = 1; Console.WriteLine("Введение"); while (i <= total) { Console.WriteLine("Глава " + i); i = i + 1; } Console.WriteLine("Заключение");
План решения задачи:
  1. Написать подробное решение для нескольких тестов с помощью условий
  2. Найти похожие повторяющиеся действия
  3. Понять чем они отличаются
  4. Найти закономерность в найденных различиях
  5. Заменить закономерность на переменную
  6. Заменить условия конструкцией цикла

Переменные могут называться самыми разными словами и буквами. Из за этого людям сложно рассуждать о назначении различных переменных. Назначение переменной и её название могут не совпадать. В разных программах переменные с одним и тем же смыслом могут иметь разные названия. При обсуждении смысла переменных в циклах обычно используют два термина: счётчик и ограничение.
int i;//переменная увеличивается в цикле. это счетчик i = 1; int limit;//переменная сравнивается со счетчиком и не меняется в цикле. это ограничение. limit = 7; while (i <= limit) { print(i); i = i + 1; }
