В этой лабораторной работе будет рассмотрен способ организации и упрощения кода программы с помощью методов. В первых заданиях вам нужно будет переписать с их помощью готовые решения задач из прошлых лабораторных. Об отличиях методов высокого качества от методов низкого качества можно прочитать в главе 7 "Высококачественные методы" части II книги "Совершенный код". (открыть в новой вкладке)

Подготовка проекта

  1. Создайте новое решение Stage2 типа "консольное приложение .NET Framework". Имя проекта укажите как CodeProject.
  2. Убедитесь, что в проекте CodeProject присутствует класс Program. Он автоматически создается вместе с проектом типа "консольное приложение".
  3. Добавьте в проекте CodeProject класс Library. Он будет содержать методы для решения задач.
  4. Добавьте к решению Stage2 проект типа "Проект модульного тестирования". Назовите его TestingProject.
  5. В проекте TestingProject переименуйте файл UnitTest1 на TestSuite. Аналогично переименуйте имя класса в этом файле.
  6. Класс TestSuite будет содержать юнит-тесты для методов из класса Library. Убедитесь, что класс TestSuite помечен атрибутом TestClass.
Настройте проект TestingProject так, чтобы он мог пользоваться кодом из проекта CodeProject.
  1. В проекте TestingProject добавьте ссылку на проект CodeProject.
  2. Убедитесь, что в файле TestSuite указана директива using Microsoft.VisualStudio.TestTools.UnitTesting;
  3. В файле TestSuite добавьте директиву using CodeProject;
